# Deleting Channels

To delete a channel, you have a few options.

You can delete them on the **Channels** page, using the three button menu.

![](/img/en/category-channels14.png)

You can also delete a channel using the three button menu on the channel itself. 

![](/img/en/category-channels15.png)

# Syncing to an Existing Account

To sync a YouTube channel to an existing Odysee account, you'll first click the person icon at the top right corner (if you already have a channel, this icon will be your channel's profile picture), and click on **Channels**.

![](/img/en/category-channels1.png)

From this screen, you'll select **Sync YouTube Channel**.

![](/img/en/category-sync1.png)

# Sending AR Tips

To send a tip, you'll click the **Support** button in the channel banner. The support button is also below the video.

![Tips](https://thumbs.odycdn.com/bebfbd4cf6da73a7d4e691cbe4e9942b.webp)

## This option is also available on the content.

From this menu, you'll select the **Tip** button with the dollar icon, select an amount, and click **Send Tip**.

![Tips](https://thumbs.odycdn.com/a5b8e490d797ba077e3ffa0a449e499c.webp)

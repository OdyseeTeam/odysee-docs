# Livestream richtig einstellen

Nachdem dein Livestream erstellt wurde, wechsle zum Tab **[Lokale Einrichtung](https://odysee.com/$/livestream?t=Setup)**, dort findest du deinen Odysee Stream-Key und Streamserver.

![](/img/de/stream_key_and_stream_server_screen_dark.png#dark)
![](/img/de/stream_key_and_stream_server_screen_light.png#light)

Der Streamserver und der Streamschlüssel werden verwendet, um deiner Streaming-Software mitzuteilen, wohin dein Livestream gesendet werden soll. Kopiere diese Angaben und gebe sie bei Bedarf in die Streaming-Software ein.

## OBS

:::tip Gut zu wissen
OBS ist kostenlos, liegt direkt auf deinem Computer und erlaubt dir viele Einstellungen vorzunehmen, aus diesem Grund empfehlen wir OBS als Streaming-Software für deinen Odysee Livestream. [Zum OBS Download](https://obsproject.com/de)
:::

OBS: `Datei` > `Einstellungen` > `Stream`

1. **Plattform**: _Benutzerdefiniert_ auswählen

2. **Server**: Gebe den auf Odysee bereitgestellten Streamserver ein

3. **Streamschlüssel**: Gebe den auf Odysee bereitgestellten Streamschlüssel ein

Sofern alles korrekt ist, klicke auf **Okay**.

## Streamyard

:::note Hinweis
Um mit Streamyard auf Odysee zu streamen, wird die kostenpflichtige Basic-Version von Streamyard benötigt.
:::

Wähle auf dem [Streamyard-Hauptbildschirm](https://streamyard.com/broadcasts) nach dem Einloggen die Schaltfläche "Eine Übertragung erstellen" aus.

![](/img/de/streamyard_add_new_stream.png)

Klicke auf die Schaltfläche "Ziel hinzufügen" und dort auf "Benuutzerdefiniertes RTMP".

![](/img/de/streamyard_add_new_source.png)

Auf dem nächsten Bildschirm beginne mit der Eingabe deiner Stream-Information.

1. Unter **RTMP-Server-URL**, gebe den auf Odysee bereitgestellten Streamserver ein.

2. Unter **Stream-Key**, gebe den auf Odysee bereitgestellten Streamschlüssel ein.

3. Der Kurzname ist für dich bestimmt, du kannst hier einfach "Odysee" eingeben.

Wenn alles korrekt ist, klicke auf **RTMP-Server hinzufügen**.

![](/img/de/streamyard_add_custom_rtmp.png)

Jetzt kannst du Streamyard wie gewohnt nutzen und direkt über Streamyard live gehen.

![](/img/de/streamyard_go_live.png)

## Restream

:::note Hinweis
Um mit Restream auf Odysee zu streamen, wird die kostenpflichtige Version von Restream benötigt.
:::

Auf dem Restream-Hauptbildschirm wähle nach dem Einloggen **Stream With Studio**.

![](/img/de/restream_add_destination_arrow.png)

Sobald du im Studio bist, wähle **Add Destination** und **Custom RTMP**.

![](/img/de/restream_studio_arrow.png)

![](/img/de/restream_select_service_arrow.png)


Auf dem nächsten Bildschirm beginne mit der Eingabe deiner Stream-Information.

1. Unter **RTMP server URL**, gebe den auf Odysee bereitgestellten Streamserver ein.

2. Unter **Stream Key**, gebe den auf Odysee bereitgestellten Streamschlüssel ein.

Wenn alles korrekt ist, klicke auf **Add Channel**.

![](/img/de/restream_ender_rtmp.png)

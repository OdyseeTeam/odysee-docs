# Credits erhalten

Um die Seite anzuzeigen, klicken Sie auf das Avatar-Symbol in der oberen rechten Ecke des Bildschirms und wählen Sie die Seite aus dem Dropdown-Menü.

![](/img/en/category-monetization1.png)

Am oberen Rand des Bildschirms wird eine Liste angezeigt, in der Sie weitere Credits erhalten können:

- **Erstes Laden**: Starten Sie Ihren eigenen Kanal, indem Sie ein Video teilen.
- **Coveted Creator**: Dieser Credit wird auf der Grundlage der Gesamtzahl der verifizierten Follower (Follower mit verifizierten Konten) Ihres Kanals vergeben.
- **Die Reise**: Diese Gutschrift wird für folgende Creators vergeben.
- **Wöchentliches Ansehen**: Diese Gutschrift wird für das Ansehen von Videos an mindestens 3 Tagen in einer bestimmten Woche vergeben.
- **Favorite Five**: Diese Gutschrift wird für das Wiedereinstellen von Inhalten von 5 verschiedenen Autoren vergeben.
- **Benutzerdefinierter Code**: Wenn du einen benutzerdefinierten Code von einem Ersteller erhalten hast, kannst du ihn hier für Credits eingeben.
- **Einladungen**: Sie können Credits erhalten, indem Sie andere auf Odysee einladen. Für weitere Informationen klicken Sie bitte auf den nächsten Abschnitt.

# Einladungen

Sie können Credits erhalten, indem Sie andere zu Odysee einladen. Unter [Invites](https://odysee.com/$/invite), werden Sie einen Einladungslink erhalten.

![](/img/en/category-monetization3.png)

Sie können diese Seite auch aufrufen, indem Sie auf das Avatar-Symbol in der oberen rechten Ecke des Bildschirms klicken und im Dropdown-Menü **Einladungen** auswählen.

Geben Sie diesen Link weiter, und für jeden Nutzer, der darauf klickt, erhalten Sie ein paar Credits.

Dies ist besonders hilfreich für Creators, die ihren Odysee-Kanal mit ihren Zuschauern teilen. Anstatt den grundlegenden Link für Ihren Kanal zu veröffentlichen, geben Sie den Einladungslink an.

Bitte beachten Sie, dass ein einforderbarer Invite die gleiche Definition hat wie ein verifizierter Odysee-Nutzer. Das bedeutet, dass der geworbene Nutzer sich bei Odysee angemeldet und sein Konto verifiziert haben muss, um Credits zu erhalten. Wenn Sie "Unclaimable" sehen, bedeutet das, dass er nicht verifiziert wurde oder nicht berechtigt ist.

# Credits empfangen und versenden

## Versenden

Um Credits von der Brieftasche aus zu versenden, klicke auf die Schaltfläche **Senden**, die unten zu finden ist.

![](/img/en/category-wallet5.png)

Von hier aus kannst du eine direkte Adresse eingeben. Wenn du eine Credits-Brieftasche außerhalb von Odysee hast, kannst du die Adresse hier einfügen.

![](/img/en/category-wallet6.png)

Alternativ kannst du auch andere kreative Geister innerhalb von Odysee suchen und mit Credits unterstützen.

## Empfangen

Um Credits zu empfangen, klicke auf die Schaltfläche **Empfangen**, die unten zu finden ist.

![](/img/en/category-wallet7.png)

Hier wird die Adresse deiner Brieftasche angezeigt. Wenn dir jemand Credits an diese Adresse sendet, erscheint es in deiner Brieftasche auf Odysee.

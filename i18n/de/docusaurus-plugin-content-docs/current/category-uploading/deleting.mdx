# Uploads löschen

Um einen Upload vollständig zu löschen, klicke auf das Mülleimer-Symbol <OdyseeIcon type="trash"/> unter einem deiner Uploads..

<Image src="/img/de/content_interaction_arrow_to_trash_dark.png#dark" />
<Image src="/img/de/content_interaction_arrow_to_trash_light.png#light" />


Du kannst auch im Drei-Punkte-Menü <OdyseeIcon type="morevertical"/> auf **<OdyseeIcon type="trash"/> Löschen** klicken:

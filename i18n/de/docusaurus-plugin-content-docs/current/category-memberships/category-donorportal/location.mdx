# Unterstützer-Menü finden

Du kannst auf die [Untersützer-Seite](https://odysee.com/$/memberships/supporter) über die [Seite für Unterstützer](https://odysee.com/$/memberships) zugreifen. Klicke dafür auf das Profilbild-Symbol <OdyseeButtonRound><OdyseeIcon type="person"/></OdyseeButtonRound> in der oberen rechten Ecke und dann auf [<OdyseeIcon type="membership"/> Mitgliedschaften](https://odysee.com/$/memberships).

<Image img="/img/de/profile_menu_locate_memberships_dark.png#dark" />
<Image img="/img/de/profile_menu_locate_memberships_light.png#light" />

Nachdem du zur Startseite für Mitgliedschaften navigiert hast, siehst du zwei verschiedene Optionen.

![](https://i.imgur.com/1IBQMGD.png)

Um die Kanalmitgliedschaften zu verwalten, denen du beigetreten bist, klicke die Schaltfläche <PrimaryOdyseeButton url="https://odysee.com/$/memberships/supporter">Unterstützer-Menü öffnen</PrimaryOdyseeButton>.

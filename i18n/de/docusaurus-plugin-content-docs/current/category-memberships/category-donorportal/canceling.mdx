# Mitgliedschaft kündigen

Um eine Mitgliedschaft zu kündigen, navigiere zu dem Kanal, dem du beigetreten bist, und wähle die Registerkarte **Mitgliedschaften** aus.

<Image img="/img/de/membership_page_bought_dark.png#dark" />
<Image img="/img/de/membership_page_bought_light.png#light" />

Du kannst auch über das Unterstützer-Menü zur Kanalseite navigieren.

<Image img="/img/de/donor_page_overview_arrow_to_page_dark.png#dark" />
<Image img="/img/de/donor_page_overview_arrow_to_page_light.png#light" />

Klicke danach einfach auf das Zahnradsymbol <OdyseeButtonRound><OdyseeIcon type="cogwheel"/></OdyseeButtonRound> und wähle dort **Mitgliedschaft kündigen** aus. Danach wird deine Mitgliedschaft gekündigt.

<Image img="/img/de/membership_page_cancel_membership_dark.png#dark" />
<Image img="/img/de/membership_page_cancel_membership_light.png#light" />

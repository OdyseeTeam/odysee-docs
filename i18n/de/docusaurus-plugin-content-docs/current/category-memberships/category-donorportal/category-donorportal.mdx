# Donor Portal

Möchten Sie Mitglied Ihres Lieblings-Creators werden? Dann sind Sie hier richtig.

:::info Wichtig

Bei der Umstellung von Odysee auf das neue Zahlungssystem sind drei Dinge zu beachten:

:::

1. Sobald Sie [sich für eine Brieftasche anmelden](https://help.odysee.tv/de/category-monetization/setup), sie haben eine Woche Zeit, um Zahlungen für bestehende Mitgliedschaften zu leisten.
2. Aufgrund der Wallet-Struktur und des dezentralen Systems müssen Sie monatliche Zahlungen leisten, um Ihre Mitgliedschaft aktiv zu halten. Dies ist frühestens zwei Wochen vor dem Verlängerungsdatum möglich.
3. Sie können jetzt niedrigere Mitgliedschaftsstufen mit einem Mindestbetrag von $0.10 einrichten. Sie können dies sogar für bestehende Stufen mit Mitgliedern aktualisieren - deren nächste Zahlung entspricht der aktualisierten Zahlung. Sie können den Preis jedoch nicht höher ansetzen.

# Kanalmitgliedschaften finden

Du kannst auf die [Kanalmitgliedschaften-Seite](https://odysee.com/$/memberships/creator) über die [Seite für Mitgliedschaften](https://odysee.com/$/memberships) zugreifen. Klicke dafür auf das Profilbild-Symbol <OdyseeButtonRound><OdyseeIcon type="person"/></OdyseeButtonRound> in der oberen rechten Ecke und dann auf [<OdyseeIcon type="membership"/> Mitgliedschaften](https://odysee.com/$/memberships).

![memberships](https://thumbs.odycdn.com/f458f5cfc1a5e17b24f7a5370337b018.webp)

Nachdem du zur Startseite für Mitgliedschaften navigiert hast, siehst du zwei verschiedene Optionen.

![memberships](https://thumbs.odycdn.com/c8e5e4ef94d294330f0e9a18784fc661.webp)

Um deine eigenen Stufen als Ersteller zu verwalten, klicke dort auf die Schaltfläche <PrimaryOdyseeButton url="https://odysee.com/$/memberships/creator">Mitgliedschaften verwalten</PrimaryOdyseeButton>.

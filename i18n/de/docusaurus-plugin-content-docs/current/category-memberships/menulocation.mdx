# Menü-Navigation

Du kannst sowohl das Unterstützer-, als auch das Mitgliedschaften-Portal über die Seite Meine Mitgliedschaften ansteuern. Klicke auf dein Avatar-Symbol in der oberen rechten Ecke und dann auf Meine Mitgliedschaften.

# Zurücksetzen des Passworts

Wenn du dein Passwort vergessen hast, ist deine erste Option, über die [Anmeldeseite](https://odysee.com/$/signin) auf **Passwort vergessen?** zu klicken.

Von hier aus kannst du deine E-Mail-Adresse eingeben und über  **Passwort zurücksetzen** einen Link zum Zurücksetzen deines Passworts anfordern.

![](/img/de/password_reset_dark.png#dark)
![](/img/de/password_reset_light.png#light)

Folge dem Link in der E-Mail, um ein neues Passwort festzulegen.

![](/img/de/set_new_password_dark.png#dark)
![](/img/de/set_new_password_light.png#light)

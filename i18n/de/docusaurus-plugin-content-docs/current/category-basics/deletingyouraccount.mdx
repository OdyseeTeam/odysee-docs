# Konto löschen


Um eine Löschanfrage für dein Konto zu stellen, lösche zuerst deine [Kanäle](/category-channels/deletingchannels) und [Uploads](/category-uploading/deleting).
Danach kannst du dein Konto über unseren E-Mail Support über hello@odysee.com löschen lassen.

:::info

Derzeit können Konten nur über unseren E-Mail Support gelöscht werden.

:::

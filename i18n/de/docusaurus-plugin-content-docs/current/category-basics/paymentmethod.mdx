# Kreditkarte hinzufügen

Indem du eine Kredit- oder Debitkarte hinzufügst, kannst du kreative Geister auf Odysee finanziell unterstützen, Videos vorbestellen und vieles mehr.

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs groupId="device">
  <TabItem value="web" label="Web">
    Um eine Karte hinzuzufügen, musst du zuerst deine <OdyseeButton url="https://odysee.com/$/wallet"><OdyseeIcon type="credits" />  Brieftasche</OdyseeButton> öffnen. Deine Brieftasche befindet sich oben rechts in der Menüleiste.
    <Image img="/img/de/locate_wallet_desktop_dark.png#dark" />
    <Image img="/img/de/locate_wallet_desktop_light.png#light" />

  </TabItem>
  <TabItem value="mobile" label="Mobile">
    Um eine Karte hinzuzufügen, gehe zuerst in deine <a href="https://odysee.com/$/wallet"><OdyseeIcon type="wallet" />  Brieftasche</a>.
    In der mobilen Version findest du die Brieftasche im Seitenleistenmenü.
    <Image img="/img/de/locate_wallet_mobile_dark.png#dark" />
    <Image img="/img/de/locate_wallet_mobile_light.png#light" />
  </TabItem>
</Tabs>

Wähle hier <OdyseeButton url="https://odysee.com/$/settings/card"><OdyseeIcon type="cogwheel" /> Zahlungsmethoden</OdyseeButton> aus.

<Tabs groupId="device">
  <TabItem value="web" label="Web">
    <Image  img="/img/de/wallet_screen_arrow_to_payment_methods_desktop_dark.png#dark" />
    <Image img="/img/de/wallet_screen_arrow_to_payment_methods_desktop_light.png#light" />
  </TabItem>
  <TabItem value="mobile" label="Mobile">
    <Image img="/img/de/wallet_screen_arrow_to_payment_methods_mobile_dark.png#dark" />
    <Image img="/img/de/wallet_screen_arrow_to_payment_methods_mobile_light.png#light" />
  </TabItem>
</Tabs>

Auf dem Bildschirm „[Zahlungsmethoden](https://odysee.com/$/settings/card)“ gibst du deinen vollständigen Namen, die Kartennummer, das Ablaufdatum und den dreistelligen CVC-Code ein.

<Image src="/img/de/add_credit_card_dark.png#dark" />
<Image src="/img/de/add_credit_card_light.png#light" />

Wenn du bereits eine Karte hinzugefügt hast, wird diese auf dem Bildschirm „[Zahlungsmethoden](https://odysee.com/$/settings/card)“ angezeigt.

<Image src="/img/de/credit_card_added_dark.png#dark" />
<Image src="/img/de/credit_card_added_light.png#light" />

Von hier aus kannst du **die Karte entfernen**, **den Transaktionsverlauf ansehen** und **deine Währung auf US-Dollar oder Euro einstellen**.

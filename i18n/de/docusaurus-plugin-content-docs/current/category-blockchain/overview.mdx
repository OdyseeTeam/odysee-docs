# Überblick

## LBRY

Die Odysee-Plattform basiert auf einem Blockchain-Protokoll namens LBRY. Es ist wichtig, sich darüber im Klaren zu sein, dass Odysee und LBRY nicht dasselbe sind. LBRY ist ein *Protokoll*, und Odysee ist eine *Plattform*, die auf diesem Protokoll aufbaut.

Die häufigste Anwendung für Blockchains ist die Kryptowährung. Das ist nicht der Verwendungszweck von LBRY bei Odysee.

Der Zweck der LBRY-Blockchain steckt im Namen *Library* - es ist eine digitale Medienbibliothek. LBRY existiert als dezentralisierte Möglichkeit, digitale Medieninhalte zu speichern und zu teilen.

Jedes Mal, wenn du eine Datei auf Odysee hochladen möchtest, dann wird diese in Wahrheit in das LBRY-Netzwerk hochgeladen, die Odysee dann abruft.

Das bedeutet, dass jeder Kanal, jeder Beitrag, jeder Kommentar und so weiter nicht nur über Odysee, sondern über jedes Frontend für das LBRY-Protokoll zugänglich ist.

Odysee hat Gemeinschaftsichtlinien und wir entfernen Inhalte, die gegen diese Richtlinien verstoßen. Das bedeutet, dass wir die Inhalte nicht mehr von LBRY abrufen. Allerdings können diese Inhalte nie vollständig aus dem LBRY-Netzwerk entfernt werden. Das bedeutet, wenn Inhalte von Odysee gesperrt werden, können diese Inhalte immer noch auf anderen LBRY-Frontends angezeigt werden, vorausgesetzt, dass diese Frontends sie nicht ebenfalls gesperrt haben.

## LBRY-Credits (LBC)

LBRY wird nicht als Kryptowährung verwendet, aber es gibt einen eigenen Token namens LBC (LBRY-Credits). Dieser Token ist ein sogenannter *Utility-Token*, d.h. er hat einen bestimmten Zweck innerhalb von LBRY und wird für seine *Nutzung* und nicht als Finanzinvestition bewertet.

Vereinfacht ausgedrückt sind es LBC, mit denen Inhalte auf der Blockchain hinterlegt werden können. LBC werden für Uploads benötigt, denn ohne sie ist das Hochladen auf LBRY buchstäblich unmöglich.

# Einführung

Eines der Hauptunterscheidungsmerkmale von Odysee gegenüber anderen Plattformen ist, dass Odysee auf einer Blockchain namens LBRY basiert.

In diesem Abschnitt werden wir untersuchen, was genau das bedeutet und wie Odysee von der Blockchain beeinflusst wird. Es ist wichtig zu erwähnen, dass niemand etwas über Blockchains wissen muss, um Odysee zu nutzen. Es ist ähnlich wie beim Autofahren, man kann ein Auto fahren, ohne zu verstehen, wie der Motor funktioniert.

Dieser Abschnitt wird sich hauptsächlich darauf konzentrieren, wie Odysee mit dem LBRY-Protokoll interagiert. Für einen detaillierteren Blick auf alle Aspekte des LBRY-Protokolls bitte https://lbry.com/faq besuchen.

# YouTube-Sync Einleitung

Das Hochladen von Hunderten alter Videos auf eine neue Plattform kann eine entmutigende Aufgabe sein. Glücklicherweise verfügt Odysee über ein [YouTube-Synchronisierungstool](https://odysee.com/$/youtube?reset_scroll=youtube&new_channel=true), mit dem YouTuber alle vergangenen und zukünftigen Inhalte automatisch auf ihren eigenen Odysee-Kanal spiegeln können.

Hier gehen wir auf die Grundlagen der Synchronisierung selbst ein und beantworten häufig gestellte Fragen.

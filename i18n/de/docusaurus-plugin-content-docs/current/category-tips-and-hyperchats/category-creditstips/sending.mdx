# Tipps zu Credits senden

Um einen Tipp zu senden, klicken Sie auf die Schaltfläche **Support** unter einem Upload.

![](/img/en/category-monetization6.png)

Diese Option ist auch auf den Kanälen verfügbar.

![](/img/en/category-monetization8.png)

In diesem Menü wählen Sie die Schaltfläche **Tipp** mit dem Credits-Symbol, wählen einen Betrag und klicken auf **Tipp senden**.

Dieser Credits-Tipp wird den getippten Inhalt/Kanal verstärken, bis der Ersteller beschließt, ihn für die Verwendung an anderer Stelle auf Odysee freizugeben.

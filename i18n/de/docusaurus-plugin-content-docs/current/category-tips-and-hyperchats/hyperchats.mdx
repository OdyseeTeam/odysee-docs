# Hyperchats & Kommentar-Tipps

## Hyperchats

Hyperchats sind hervorgehobene Chats während Livestreams. Sie werden gesendet, indem ein Tipp an den Chat angehängt wird.

Nachdem Sie Ihren Chat eingegeben haben, können Sie einen Hyperchat senden, indem Sie entweder das Credits-Symbol oder das Dollarzeichen unter dem Chatfeld auswählen.

Hier wählen Sie einen Betrag aus und klicken auf **Senden**.

Ihr Hyperchat wird hervorgehoben und zeigt den getippten Betrag an.

## Kommentar-Tipps

Kommentar-Tipps werden gesendet, indem Sie einen Tipp an einen Kommentar anhängen.

Nachdem Sie Ihren Kommentar geschrieben haben, können Sie einen Hyperchat senden, indem Sie entweder das Credits-Symbol oder das Dollarzeichen unter dem Kommentarfeld auswählen.

Hier wählst du einen Betrag aus und drückst auf **Senden**.

In Ihrem Kommentar wird der Trinkgeldbetrag angezeigt.

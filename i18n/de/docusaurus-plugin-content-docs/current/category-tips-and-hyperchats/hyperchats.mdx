# Hyperchats & Kommentar-Tipps

## Hyperchats

Hyperchats sind hervorgehobene Chats während Livestreams. Sie werden gesendet, indem ein Tipp an den Chat angehängt wird.

Nachdem Sie Ihren Chat eingegeben haben, können Sie einen Hyperchat senden, indem Sie entweder das Dollar-Symbol unter dem Chat-Feld auswählen.

Von hier aus wählst du einen Betrag aus und drückst auf **Senden**.

Ihr Hyperchat wird hervorgehoben und zeigt den Trinkgeldbetrag an.

![hyperchats](https://thumbs.odycdn.com/5cb27970680ea458b6704ce98729c20f.webp)

## Kommentar-Tipps

Kommentar-Tipps werden gesendet, indem Sie einen Tipp an einen Kommentar anhängen.

Nachdem Sie Ihren Kommentar geschrieben haben, können Sie einen Hyperchat senden, indem Sie entweder das Dollar-Symbol unter dem Kommentarfeld auswählen.

Hier wählst du einen Betrag aus und drückst **Senden.**.

In Ihrem Kommentar wird der Betrag des Trinkgelds angezeigt.

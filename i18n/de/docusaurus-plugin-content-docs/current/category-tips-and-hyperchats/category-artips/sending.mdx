# AR-Tipps senden

Um einen Tipp zu senden, klicken Sie auf die Schaltfläche **Support** auf dem Kanalbanner. Die Schaltfläche **Support** befindet sich auch unter dem Video.

![Tips](https://thumbs.odycdn.com/bebfbd4cf6da73a7d4e691cbe4e9942b.webp)

## Diese Option ist auch für Inhalte und Kanäle verfügbar.

Wählen Sie in diesem Menü die Schaltfläche **Tipp** mit dem Dollar-Symbol, wählen Sie einen Betrag und klicken Sie auf **Tipp senden**.

![Tips](https://thumbs.odycdn.com/a5b8e490d797ba077e3ffa0a449e499c.webp)

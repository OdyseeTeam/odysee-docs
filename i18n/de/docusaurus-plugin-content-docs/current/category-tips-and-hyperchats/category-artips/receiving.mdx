# Empfang von AR-Tipps

Wenn du einen Tipp auf AR erhältst, erhältst du die Unterstützung der Community für die Inhalte deines Kanals. Dieser Tipp wird direkt auf Ihre Odysee-Geldbörse übertragen. Sie können alle erhaltenen Tipps auf Ihrer Walletseite einsehen.

![wallet](https://thumbs.odycdn.com/e6e0da12567980d225af59fdfb317f86.webp)

Auf diesem Bildschirm wählen Sie **Bilanz** und wechseln zur Registerkarte **Tipp**. Dort sehen Sie alle Tipps, die Sie erhalten haben.

![Tips](https://thumbs.odycdn.com/04649ce7117cffdf0fcc6a6859401177.webp)

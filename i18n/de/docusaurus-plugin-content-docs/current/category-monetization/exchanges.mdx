Arweave ist ein dezentrales Speicherprotokoll, mit dem Daten dauerhaft auf der Blockchain gespeichert werden können. Odysee nutzt den Arweave-eigenen **AR**-Token, um Zahlungen, Tipps oder Mitgliedschaften auf Odysee zu ermöglichen.

Um Ihre **AR-Token** zu verwalten, verwenden Sie die AR-Wallet auf Odysee oder [Wander wallet](https://www.wander.app/) (verfügbar als Browser-Erweiterung und mobile App). Wander ist hier zu bevorzugen, da es Ihnen eine sichere und einfache Interaktion mit dem Arweave-Netzwerk ermöglicht.

:::info Wichtig

Wenn Sie noch keine Brieftasche erstellt oder Wander installiert haben, empfehlen wir Ihnen, die Dokumentationsseite zu besuchen, um eine Schritt-für-Schritt-Anleitung zu erhalten, wie Sie eine Brieftasche erstellen oder Wander installieren.

:::

In diesem Tutorial zeigen wir Ihnen, wie Sie Ihre AR-Token von Ihrem Wander-Wallet an eine Börse wie Binance, Kraken, Crypto.com oder Coinex senden, um sie zu verkaufen oder zu handeln.

## Schritte zum Senden von AR an eine Börse

1. **Stellen Sie sicher, dass Sie AR in Ihrem Wallet haben:** Öffnen Sie Ihr AR-Wallet, indem Sie auf Odysee oder über Ihr Wander-Wallet (Erweiterung oder App) auf „Arweave-Konto“ klicken. Stellen Sie sicher, dass Sie genügend AR für die Transaktion haben (berücksichtigen Sie dabei Netzwerkgebühren).

![Wallet](https://thumbs.odycdn.com/0b4cda34b94d09b481d46bda5b37de92.webp)

2. **Beschaffen Sie sich die AR-Einzahlungsadresse der Börse:** Melden Sie sich bei Ihrem Binance-, Kraken-, Coinex- oder anderen AR-fähigen Börse-Konto an. Gehen Sie zum Bereich „Wallet“ oder „Einzahlungen“. Suchen Sie nach Arweave (AR) und kopieren Sie die angegebene Einzahlungsadresse.

3. **Auf der Wallet-Seite in Odysee:** Klicken Sie in Ihrem Odysee-Wallet auf die Schaltfläche **„Senden“**.

4. **Richten Sie die Transaktion in der Wander-Erweiterung ein:** Klicken Sie in Wander auf „Senden“. Fügen Sie die Börsenadresse in das Feld „Ziel“ ein. Geben Sie den zu sendenden AR-Betrag ein.

:::info Wichtig

Überprüfen Sie, ob die Adresse korrekt ist (Börsen erfordern möglicherweise eine bestimmte Adresse für AR).

:::

5. **Warten Sie auf die Netzwerkbestätigung:** Transaktionen auf Arweave werden in der Regel innerhalb weniger Minuten bestätigt. Sie können den Status Ihrer Transaktion in einem Block-Explorer wie überprüfen [ViewBlock](https://viewblock.io/arweave).

6. **Überprüfen Sie die Einzahlung an der Börse:** Die Gutschrift kann einige Minuten dauern. Überprüfen Sie den Einzahlungsverlauf in Ihrem Börsenkonto.

## Liste der unterstützten Börsen.

Weitere Informationen zu AR-kompatiblen Börsen finden Sie in der vollständigen Liste auf [CoinMarketCap](https://coinmarketcap.com/currencies/arweave/).

1. Binance - AR/USDT.
2. Bybit - AR/USDT.
3. OKX - AR/USDT.
4. Crypto,com - AR/USD.
5. Kraken - AR/USD.
6. MEXC - AR/USDT.
7. KuCoin - AR/USDT.
8. CoinEx - AR/USDT.

## Wichtige Tipps

✅ Überprüfen Sie vor dem Senden immer die Zieladresse.
✅ Senden Sie zunächst einen kleinen Betrag zum Testen.
✅ Stellen Sie sicher, dass die Börse AR-Einzahlungen im Arweave-Netzwerk unterstützt.

**Achtung! Verwenden Sie keine anderen Netzwerke wie ERC-20.**

Fertig! Jetzt wissen Sie, wie Sie Ihre AR-Token von Wander an Ihre bevorzugte Börse senden.

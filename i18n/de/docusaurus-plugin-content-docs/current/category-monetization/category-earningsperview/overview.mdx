# Vergütung für Aufrufe

Mit Zuschauerprämien erhalten kreative Geistern LBC für jeden verifizierten View bzw. Ansicht. Ein verifizierter View ist ein View von einem verifizierten Nutzer. Das bedeutet, dass wir verifiziert haben, dass es sich um eine echte Person handelt.Um Missbrauch zu verhindern, werden nur die ersten sieben Views eines verifizierten Nutzers pro Tag vergütet.

## Wie viel verdiene ich pro Ansicht?

Um deine Einnahmen pro View bzw. Ansicht zu überprüfen, navigiere zur Seite Kanäle, indem du das Profilbildsymbol <OdyseeButtonRound><OdyseeIcon type="person"/></OdyseeButtonRound> anklickst und im Dropdown-Menü **[<OdyseeIcon type="atsign"/> Kanäle](https://odysee.com/$/channels)** auswählst.

![](/img/de/homepage_profile_menu_open_arrow_to_channels_desktop_dark.png#dark)
![](/img/de/homepage_profile_menu_open_arrow_to_channels_desktop_light.png#light)

Deine Einnahmen pro View/Ansicht werden rechts neben deinem Kanal angezeigt.

![](/img/de/channel_overview_screen_arrow_dark.png#dark)
![](/img/de/channel_overview_screen_arrow_light.png#light)

:::info Hinweis
Die <OdyseeIcon type="credits"/> Vergütung pro Aufruf kann sich ändern und basiert auf einer Reihe von Variablen, wie z. B. der durchschnittlichen Betrachtungsdauer, der durchschnittlichen Anzahl der Aufrufe, der Art des Inhalts, dem Engagement, dem Standort des Erstellers und dem Preis von LBC.
:::

## Einnahmen verfolgen

Klicke auf dein Profilbildsymbol <OdyseeButtonRound><OdyseeIcon type="person"/></OdyseeButtonRound> in der oberen rechten Ecke auf Odysee und wähle dort [<OdyseeIcon type="Award"/> Prämien](https://odysee.com/$/rewards).

Auf dieser Seite wird deine Vergütung für jeden Monat unter **Erhaltene Prämen** mit der Bezeichnung *Content View Tips* angezeigt.

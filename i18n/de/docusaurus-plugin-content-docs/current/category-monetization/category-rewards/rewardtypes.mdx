# Prämien

Die Prämienseite kannst du über dein Profilbild am oberen rechten Rand aufrufen. Im Dropdown-Menü dann **Prämien** auswählen.

Im oberen Teil des Bildschirms wird die Liste der angebotenen Belohnungen angezeigt. Diese sind:

|Prämie| Beschreibung |
|:-|:-|
| **Meister der Aufrufe** | Diese Prämie wird für angesehene Videos auf Odysee vergeben. Um Missbrauch zu vermeiden, werden nur 10 Videos pro Tag gezählt. |
| **Begehrter Creator** | Diese Prämie wird für verifizierte Follower (Follower mit verifizierten Konten) deines Kanals vergeben. |
| **Die Reise** | Diese Prämie wird für das Abonnieren von Kanälen vergeben. |
| **Wöchentliche Zuschauerprämie** | Diese Prämie gibt es für das Ansehen von Videos an mindestens drei Tagen in einer bestimmten Woche. |
| **Fünf Lieblinge**  | Diese Belohnung gibt es für das Reposten von Inhalten von fünf verschiedenen Kanälen. |
| **Benutzerdefinierter Code** | Wenn du einen benutzerdefinierten Reward-Code von einem Ersteller erhalten hast, kannst du ihn hier eingeben. |
| **Einladungen** | Du kannst auch LBC verdienen, indem du Leute zu Odysee einlädst. Für weitere Informationen zu **Einladungen** klicke bitte auf den nächsten Abschnitt. |

# Hyperchats & Kommentar-Trinkgelder

## Hyperchats

Hyperchats sind hervorgehobene Chats während Livestreams. Sie werden gesendet, indem ein Tipp an den Chat angehängt wird.

Wenn du ein Hyperchat verschicken willst, musst du während eines Livestreams einen Text eintippen und daraufhin entweder das LBC-Symbol oder das Dollarzeichen unter dem Chatfeld auswählen.

Wähle einen Betrag aus und klicke auf **Senden**.

Dein Hyperchat wird hervorgehoben und zeigt den Betrag des Trinkgelds an.

## Kommentar-Trinkgelder

Kommentar-Trinkgelder werden verschickt, indem du das Trinkgeld an einen Kommentar unter einem Video anheftest. 

Nachdem du einen Kommentar geschrieben hast, kannst du entweder das LBC-Symbol oder das Dollarzeichen unter dem Kommentarfeld auswählen.

Wähle einen Betrag aus und klicke auf **Senden**.

In deinem Kommentar wird der Betrag angezeigt, den du als Trinkgeld gegeben haben.

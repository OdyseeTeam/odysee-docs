# Playlisten veröffentlichen

Um deine Playlisten für andere sichtbar zu machen, muss sie zunächst veröffentlicht werden.

Um eine Playlist zu veröffentlichen, navigieren zum Menü <a href="https://odysee.com/$/playlists"><OdyseeIcon type="Playlist"/> Playlisten</a>, indem du in der Seitenleiste auf die Schaltfläche **Playlisten** klickst.

Bewege den Mauszeiger über die Playliste, die du veröffentlichen möchtest, klicke auf das Drei-Punkte-Menü <OdyseeIcon type="MoreVertical"/> auf der rechten Seite und wähle **<OdyseeIcon type="uploadCloud"/> Veröffentlichen** aus.

Hier kannst du auch Thumbnails, Tags oder eine Beschreibung hinzufügen, aber alles, was benötigt wird, sind ein Name und ein ausgewählter Kanal. 

Wenn du den Namen und Kanal ausgewählt hast, auf dem du die Playlist veröffentlichen möchtest ausgewählt hast, klicke auf <PrimaryOdyseeButton>Senden</PrimaryOdyseeButton>.

Nachdem die Veröffentlichung bestätigt wurde, erscheint die Playlist auf deinem Kanal unter der Registerkarte Playlisten. Dort kann sie von jedermann eingesehen werden.

:::info Hinweis
Damit wird die aktuelle Version deiner Playlist veröffentlicht. Wenn du Änderungen an der Playlist vornimmst, musst du sie erneut veröffentlichen, damit die Änderungen öffentlich gemacht werden.
:::

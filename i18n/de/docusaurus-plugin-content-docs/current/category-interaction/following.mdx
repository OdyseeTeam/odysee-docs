# Abonnieren

Gibt es einen kreativen Odysee-Geist, dessen Inhalte du besonders magst? Du kannst ihm oder ihr folgen, in dem du auf die Schaltfläche **Abonnieren** auf ihrem Kanal oder unter ihren Inhalten klickst. 

<Image src="/img/de/channel_page_subscribe_arrow_desktop_dark.png#dark" />
<Image src="/img/de/channel_page_subscribe_arrow_desktop_light.png#light" />

Immer wenn du einen Kanal abonnierst, wird dieser unter **[Meine Abos](https://odysee.com/$/following)** hinzugefügt. Videos unter "Meine Abos" werden standardmäßig nach dem Veröffentlichunsdatum sortiert. 

<Image src="/img/de/how_to_manage_followers_dark_de.png#dark" />
<Image src="/img/de/how_to_manage_followers_light_de.png#light" />

Um eine Liste deiner abonnierten Kanäle anzuzeigen, klicke auf das [<OdyseeIcon type="cogwheel"/>](https://odysee.com/$/following/manage) Zahnrad neben [Meine Abos](https://odysee.com/$/following/manage) in der erweiterten Seitenleiste.

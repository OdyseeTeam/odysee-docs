# Canceling a Membership

To cancel a membership, navigate to the channel you've joined and select the membership tab.

![](https://i.imgur.com/vjnKkRD.png)

You can also navigate to this screen through the Donor portal.

Simply click the gear icon and select Cancel Membership, and your membership will be canceled.

![](https://i.imgur.com/0pZGd6T.png)

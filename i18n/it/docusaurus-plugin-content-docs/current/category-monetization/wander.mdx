# Wallet Wander Integrato

Il modo più semplice per usare il sistema di pagamenti di Odysee è [creare un account](https://help.odysee.tv/it/category-monetization/setup) direttamente con Wander, integrato in Odysee. Questo creerà un wallet collegato al tuo metodo di accesso e un file di backup personalizzato che sarà necessario anche per l’uso su altri browser, dispositivi, mobile, ecc. Dovrai anche eseguire un backup completo del wallet nel caso in cui tu perda l’accesso al login o al dispositivo.

## File di recupero

Il file di recupero viene utilizzato insieme al tuo accesso Wander per aiutarti a ripristinare il wallet in caso di perdita di dati locali o su altri dispositivi dove usi lo stesso wallet. Questo file da solo non può essere usato per accedere al wallet, quindi puoi conservarlo in modo sicuro anche nel cloud per un accesso semplice. Tuttavia, avrai bisogno dell’accesso al tuo account Wander per usarlo insieme a questo file di recupero.

Puoi trovare maggiori informazioni sul [sito di Wander](https://www.wander.app/help/what-is-a-wander-connect-recovery-file).

![wallet](https://thumbs.odycdn.com/1971f33e3acacd51805ce51a8e3b257e.webp)

## Fai il backup del tuo wallet: File chiave o frase seed

**Chiavi private:** Assicurati di proteggerle adeguatamente e di non conservarle nel cloud. Possono essere usate per ripristinare il wallet senza accesso al login.

Dopo aver collegato e creato il wallet con Wander, dovrai fare il backup della frase di recupero. Di seguito puoi vedere alcune delle opzioni disponibili per il backup. Puoi scaricare il file di recupero, esportare il wallet o generare un codice QR.

La tua frase di sicurezza (**o frase seed**) è la chiave principale del wallet. Se perdi l’accesso al dispositivo, queste 12 o 24 parole saranno l’unico modo per recuperare i tuoi fondi. È fondamentale conservarle in un luogo sicuro e offline.

Se decidi di conservare il file chiave, assicurati di proteggerlo con una password nel caso qualcuno ottenga accesso non autorizzato.

:::info Importante

Annotale su carta e conservale in un luogo sicuro, come un gestore di password tipo **Bitwarden**, oppure usa un dispositivo di archiviazione cifrato. Non condividerle mai con nessuno e non conservarle nel cloud.

:::

![wallet](https://thumbs.odycdn.com/69c71c30f37b9d4b50195e40ec1ebfa3.webp)

## E ora?

1. **Aggiungi fondi al wallet:** Aggiungi criptovalute compatibili per pagamenti singoli, mance, membership o acquisti. Questo sistema riduce le commissioni di transazione e permette micro-pagamenti non possibili con processori tradizionali come Stripe.
2. **Invia e ricevi pagamenti:** Usa i fondi aggiunti per supportare i creator, ricevere mance e inviare pagamenti a chiunque, ovunque.
3. **Gestisci membership:** Paga le membership in modo flessibile pre-caricando il wallet.
4. **Preleva i tuoi guadagni:** Converti i guadagni in euro o altre criptovalute tramite piattaforme esterne come Kraken o Binance.

:::info Importante

Per maggiori informazioni sugli exchange disponibili e su come convertire i tuoi guadagni, visita la [pagina degli exchange qui](https://help.odysee.tv/it/category-monetization/exchanges).

:::

## Estensione Wander Wallet e app Wander

Un’altra opzione per un wallet Arweave, oltre alla versione integrata in Odysee, è l’estensione e l’app mobile di Wander. Disponibile per iOS, Android e come estensione browser, Wander Wallet ti permette di gestire il token AR in modo più sicuro direttamente su Odysee. [Scarica Wander per iniziare](https://www.wander.app/). Consigliamo questa opzione se hai esperienza con wallet crypto o sei un utente esperto.

## Come creare un wallet con Wander?

![Wander](https://thumbs.odycdn.com/a556307b756714f7f557b24ed8c84176.webp)

## 1. Imposta la password di accesso al wallet

Il primo passo per creare il wallet è impostare una password di accesso. Questa password sarà richiesta ogni volta che usi il wallet nel browser. Aggiunge un livello essenziale di sicurezza e, se la dimentichi, puoi reimpostarla usando la frase seed, un backup che vedremo nel prossimo passo.

![Wander](https://thumbs.odycdn.com/b95dd8766a8e9046e156cbaa6cf2c5eb.webp)

## 2. Abilita i permessi

Per una migliore esperienza, abilita i seguenti permessi. Potrai sempre modificarli in seguito.

![Wander](https://thumbs.odycdn.com/d5e179a96bc28aa49d67bd0312cde654.webp)

## 3. Completa l’installazione di Wander

Una volta creata la tua wallet, dovrai seguire una breve panoramica della wallet e configurare la sicurezza e le opzioni di backup.

![Wander](https://thumbs.odycdn.com/a3ecf90f44ecaa6ed936dcce89e646e3.webp)

## 4. Proteggi la tua frase seed

La frase seed è la parte più importante durante la configurazione della tua wallet. Consiste in 12 parole generate casualmente che ti permettono di recuperare la wallet su qualsiasi dispositivo. Segui queste raccomandazioni per proteggere la tua frase seed:

* Scrivila su **un foglio sicuro** oppure valuta di conservarla in un gestore di password affidabile come Bitwarden, che offre un ambiente cifrato e sicuro per informazioni sensibili.
* **Non conservarla mai in chiaro in formato digitale**, né sul telefono, computer o servizi cloud come iCloud, Google Drive, ecc.
* Tieni queste parole al sicuro. Chiunque le ottenga può prendere il controllo totale della tua wallet.
* Si consiglia di usare un gestore di password anche per il backup della chiave privata.

# Riutilizzo del server/chiave di trasmissione

Sarebbe fastidioso se dopo ogni trasmissione in diretta dovessi cambiare il server di trasmissione e la chiave di trasmissione nel tuo software di streaming. Fortunatamente, non è necessario farlo.

![](https://i.imgur.com/OPUlCNP.png)

Se hai già trasmesso in passato e prevedi di farlo di nuovo, ti basta scegliere l'opzione **Crea o programma una nuova trasmissione** e il tuo server di trasmissione e la chiave di trasmissione verranno trasferiti alla tua prossima trasmissione.

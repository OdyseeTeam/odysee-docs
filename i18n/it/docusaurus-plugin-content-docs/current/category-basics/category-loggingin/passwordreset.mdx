# Reimpostazione della password

Se non conosci la tua password, la prima opzione è cliccare su **Hai dimenticato la password?**

Da qui, inserisci nuovamente la tua email, clicca su **Reimposta password** e ti invieremo un'email per reimpostare la password.

![](/img/es/image_22.png)

Segui il link nell'email per impostare una nuova password.

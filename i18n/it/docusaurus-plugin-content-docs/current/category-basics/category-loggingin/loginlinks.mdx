# Link magico

Se preferisci accedere a Odysee utilizzando un link inviato alla tua email invece di una password, puoi selezionare l'opzione **Usa link magico**.

![](/img/es/image_20.png)

Questo invia un'email di conferma con un link. Basta cliccare sul pulsante o sul link nell'email e completare un CAPTCHA nel tuo browser web o dispositivo, e avrai effettuato l'accesso correttamente.

:::info Informazione

Per migliorare l'esperienza di verifica, prova a utilizzare un browser compatibile e disattiva i blocchi pubblicitari o i captcha. Il link è valido solo per pochi minuti, quindi potresti dover riprovare.

:::

![](/img/es/image_21.png)

:::danger Avvertenza

Se ricevi questa email con un link magico ma non stavi tentando di accedere al tuo account, non cliccare sul link.

:::

## Ecco alcuni passaggi aggiuntivi nel caso in cui si verifichino errori imprevisti durante l'utilizzo del link magico.

1. Chiudi il tuo browser web (Firefox, Chrome, Brave).
2. Apri Odysee ed effettua l'accesso utilizzando il link magico.
3. Clicca sul link inviato alla tua email nello stesso browser.
4. Attendi qualche secondo e la tua sessione sarà attiva.

:::info Informazione

In alcuni casi, potrebbe essere necessario cancellare la cache del browser per Odysee se si verificano errori imprevisti. Inviaci un'email se riscontri un errore imprevisto a hello@odysee.com.

:::

## Perché mi viene chiesto di verificare tramite un link?

A volte, quando accedi, potresti ricevere un'email di conferma senza averla richiesta. Questo accade quando rileviamo un cambio di IP o un nuovo dispositivo. Il motivo di sicurezza dietro a ciò è evitare che persone diverse da te accedano al tuo account, anche se conoscono il tuo nome utente e la tua password.

Se cambi posizione tra gli accessi o dispositivo, ti verrà chiesto di confermare tramite email, anche se hai già usato Odysee in entrambe le posizioni in precedenza.

Questo può accadere anche se utilizzi una VPN.

# Eliminazione dell'account

Attualmente, l'eliminazione dell'account è in parte un processo manuale. Puoi eliminare tutti i contenuti e i canali del tuo account, quindi inviare una richiesta di eliminazione utilizzando l'opzione "Elimina account" nelle [Impostazioni](https://odysee.com/$/settings). Una conferma verrà inviata via email una volta che l'eliminazione sarà stata elaborata.

![](https://thumbs.odycdn.com/4d8fc2235830ac249751daa82c42bffc.webp)

:::info

Se l'opzione non è disponibile nell'app che utilizzi, elimina prima tutti i canali e i contenuti caricati che potresti avere sull'account e poi invia un'email a hello@odysee.com per richiedere l'eliminazione dell'account.

:::

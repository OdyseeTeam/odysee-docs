# Visualizza playlist

## Le tue playlist

Puoi visualizzare le tue playlist cliccando sul pulsante **Playlist** nella barra laterale.

Tutte le tue playlist, sia pubblicate che non pubblicate, appariranno nella pagina Playlist.

## Playlist di altri canali

Per visualizzare le playlist di un altro canale, accedi prima alla sua pagina canale.

Da qui, seleziona la scheda **Playlist**. Tutte le playlist pubbliche create da questo canale appariranno qui. Le playlist che non sono state rese pubbliche non appariranno qui e non saranno visibili a nessuno tranne che al canale che le ha create.

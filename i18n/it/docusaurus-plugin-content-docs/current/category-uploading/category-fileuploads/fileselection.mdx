# Caricamento e selezione dei file

Per caricare un file, fai clic sull'icona della nuvola accanto alla barra di ricerca nella parte superiore dello schermo.

![](/img/en/category-uploading1.png)

Su dispositivi mobili, questo pulsante si trova nella parte inferiore della barra laterale espansa.

![](/img/en/category-uploading2.png)

Dalla schermata di caricamento, ti verrà prima chiesto di selezionare un file. Odysee è conosciuta principalmente come piattaforma video, ma puoi anche caricare immagini, file audio, PDF, file CAD e quasi qualsiasi altra cosa.

![odysee](https://thumbs.odycdn.com/1f99600e77fc5e286c77a4831ae8d781.webp)

:::note Nota

Per i video, dovrai caricare file MP4 in formato H.264/AAC. Per la migliore esperienza di visione, consigliamo bitrate inferiori a 8 Mbps. I caricamenti su Odysee sono limitati a 16 GB.

:::

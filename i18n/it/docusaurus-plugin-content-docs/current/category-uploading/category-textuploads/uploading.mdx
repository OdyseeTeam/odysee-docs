# Posizione del menu

Per caricare un post di testo, fai clic sull'icona della penna accanto alla barra di ricerca nella parte superiore dello schermo.

![](/img/en/category-uploading12.png)

Su dispositivi mobili, questo pulsante si trova verso la parte inferiore della barra laterale espansa.

![](/img/en/category-uploading13.png)

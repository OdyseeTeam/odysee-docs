# Creating More Channels

To create another channel, you'll first want to return to the **Channels** page.

![](https://i.imgur.com/KB8nfF3.png)

From this page, you'll be able to create more channels using the **New Channel** button.

![](https://i.imgur.com/5ygxxOS.png)

You can have up to 5 channels.

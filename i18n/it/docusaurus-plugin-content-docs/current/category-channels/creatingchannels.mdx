# Crea altri canali

Per creare un altro canale, devi prima tornare alla pagina **Canali**.

![](https://i.imgur.com/A3DEdiT.png)

Da questa pagina puoi creare altri canali utilizzando il pulsante **Nuovo canale**.

![](https://i.imgur.com/xsPm12i.png)

Puoi creare fino a 5 canali. I canali sincronizzati non vengono conteggiati.

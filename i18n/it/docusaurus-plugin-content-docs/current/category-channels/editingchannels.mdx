# Modifica del canale

Per modificare una qualsiasi delle impostazioni sopra indicate, tocca l'icona della matita in basso a destra del banner del tuo canale.

![](https://i.imgur.com/PAf8ony.png)

Dopo aver apportato le modifiche, assicurati di inviarle e attendi qualche minuto affinché diventino disponibili.

![](https://i.imgur.com/Ap0fVxC.png)

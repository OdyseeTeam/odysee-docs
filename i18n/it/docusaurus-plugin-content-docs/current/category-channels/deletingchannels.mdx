# Eliminazione dei canali

Per eliminare un canale, hai diverse opzioni.

Puoi eliminarli dalla pagina **Canali**, utilizzando il menu contestuale a tre pulsanti.

![](https://i.imgur.com/345UtVh.png)

Puoi anche eliminare un canale utilizzando il menu contestuale a tre pulsanti del canale stesso.

![](https://i.imgur.com/Bha4o9U.png)

Se decidi di eliminare un canale mentre ne modifichi le informazioni, troverai l'opzione di eliminazione nel menu di modifica.

![](https://i.imgur.com/JVu5qa8.png)

:::danger ATTENZIONE

Se si verifica un problema con la sincronizzazione, non tentare di risolverlo eliminando il canale. Contattaci invece all'indirizzo help@odysee.com per risolvere il problema. Una volta eliminato, potremmo non essere in grado di risincronizzarlo o risolverlo.

:::

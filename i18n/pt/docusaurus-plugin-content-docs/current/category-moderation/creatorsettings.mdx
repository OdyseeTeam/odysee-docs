# Configurações do Criador

## Desativar comentários

A primeira configuração na página de configurações do criador é uma caixa de seleção para permitir comentários.

Se a caixa estiver marcada, seu canal terá comentários. Se a caixa não estiver marcada, seu canal terá os comentários desativados em todos os vídeos.

## Modo lento

A segunda configuração na página de configurações do construtor é *Modo Lento*. Isso define um tempo (em segundos) que os comentaristas devem esperar entre os comentários. Por exemplo, se o modo lento foi definido para 10 segundos, os comentaristas teriam que esperar 10 segundos depois de enviar um comentário para deixar outro comentário.

Isso é útil para transmissões ao vivo. O modo lento pode evitar spam e é frequentemente usado por streamers que recebem um alto volume de bate-papo ao vivo e têm problemas para acompanhar.

## Idade mínima do canal

A configuração a seguir na página de configurações do criador permite que você defina uma idade mínima do canal para comentários. Isso é útil para evitar spam e assédio.

## Dica mínima de LBC para comentários

Definir uma gorjeta mínima de LBC para comentários significa que os usuários devem dar pelo menos o valor designado para deixar um comentário. Por exemplo, se o valor mínimo da gorjeta LBC for definido como 10, os usuários precisarão dar 10 LBC ou mais para comentar.

Isso pode ser útil como forma de selecionar seções de comentários. Se alguém tiver que dar gorjeta para comentar, é provável que todos que acabem deixando um comentário sejam fãs do seu canal.

## Dica mínima de LBC para hyperchats

Definir uma gorjeta LBC mínima para hyperchats significa que os usuários devem dar gorjeta pelo menos o valor designado para conversar. Isso não afeta chats ou comentários normais de forma alguma.

## Adicionando moderadores

Em seguida é a opção de adicionar moderadores. Digite o nome do canal ou o URL da pessoa que você deseja adicionar como moderador e selecione Adicionar.

Moderadores delegados podem bloquear canais, silenciar canais e excluir comentários em seu nome.

## Filtragem de palavras

Se houver palavras específicas que você gostaria de impedir que os usuários dissessem em sua seção de comentários, você pode adicioná-las na seção Filtro.

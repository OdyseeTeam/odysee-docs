# Envio de dicas de AR

Para enviar uma dica, você clicará no botão **Suporte** no banner do canal. O botão de suporte também está abaixo do vídeo.

![Tips](https://thumbs.odycdn.com/bebfbd4cf6da73a7d4e691cbe4e9942b.webp)

## Essa opção também está disponível no conteúdo.

Nesse menu, você selecionará o botão **Tip** com o ícone de dólar, selecionará um valor e clicará em **Send Tip**.

![Tips](https://thumbs.odycdn.com/a5b8e490d797ba077e3ffa0a449e499c.webp)

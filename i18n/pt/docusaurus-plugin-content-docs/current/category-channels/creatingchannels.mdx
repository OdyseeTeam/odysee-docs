# Crie mais canais

Para criar outro canal, primeiro você deve retornar à página **Canais**.

![](https://i.imgur.com/A3DEdiT.png)

Nessa página, você poderá criar mais canais usando o botão **Novo canal**.

![](https://i.imgur.com/xsPm12i.png)

Você pode criar até 5 canais.

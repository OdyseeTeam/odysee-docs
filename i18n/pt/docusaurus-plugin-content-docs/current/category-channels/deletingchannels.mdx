# Remoção do canal

Para remover um canal, você tem algumas opções.

Você pode removê-los na página **Canais**, usando o menu de três botões.

![](https://i.imgur.com/345UtVh.png)

Você também pode remover um canal usando o menu de três botões no próprio canal.

![](https://i.imgur.com/Bha4o9U.png)

Ou, se você decidir excluir um canal enquanto estiver editando o canal, há uma opção de exclusão no menu de edição.

![](https://i.imgur.com/JVu5qa8.png)

:::danger ATENÇÃO

Se algo deu errado com a sincronização, não tente consertar excluindo o canal. Em vez disso, entre em contato conosco em help@odysee.com para corrigir o problema. Depois de excluído, talvez não consigamos sincronizá-lo novamente ou corrigi-lo.

:::

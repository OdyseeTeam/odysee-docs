# Alterações de e-mail

:::info

No momento, não temos uma maneira integrada de alterar e-mails. Podemos alterar os e-mails, mas isso deve ser feito manualmente por nós.

Se você deseja que seu e-mail seja alterado, envie um e-mail para hello@odysee.com.

:::

# Verificação

A verificação não é necessária para usar o Odysee, mas é altamente recomendada. A verificação permite que você receba alguns Créditos simplesmente usando o Odysee. Exigimos verificação para que você receba alguns Créditos, para evitar abusos.

:::info Observação

Para saber mais sobre o Créditos e seu papel na Odysee, confira nosso guia [aqui](/pt/category-blockchain/).

:::

A maioria das contas é verificada automaticamente. Dependendo de fatores como localização, uso de VPN, etc., podem ser solicitadas informações adicionais para verificação. Para fazer isso, você tem algumas opções:

![odysee](https://thumbs.odycdn.com/0f0a6d967386fc5142215b216f34db82.webp)

## Via número de telefone

Sua primeira opção é verificar através do seu telefone. Enviaremos uma mensagem de texto com um código.

## Via email

Você pode solicitar a verificação da sua conta Odysee enviando um email para hello@odysee.com.

:::info Nota

Nessa etapa, as solicitações de verificação são analisadas manualmente, o que significa que podem levar de algumas horas a vários dias para serem aprovadas.

:::

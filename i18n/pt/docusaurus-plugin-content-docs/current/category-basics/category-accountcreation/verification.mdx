# Verificação

A verificação não é necessária para usar o Odysee, mas é altamente recomendada. A verificação permite que você ganhe créditos LBRY (LBC para abreviar) simplesmente usando o Odysee. Exigimos verificação para ganhar LBC, para evitar abusos.

:::info Observação

Para saber mais sobre o LBC e seu papel na Odysee, confira nosso guia [aqui](/pt/category-blockchain/).

:::

A maioria das contas é verificada automaticamente. Dependendo de fatores como localização, uso de VPN, etc., podem ser solicitadas informações adicionais para verificação. Para fazer isso, você tem algumas opções:

**Via Phone Number**

Your first option is to verify through your phone. We'll send you a text message with a code.

**Via Cartão** 

Você pode verificar usando um cartão de crédito ou débito. Não guardamos as informações do seu cartão nem cobramos qualquer taxa. Você pode ver uma autorização de US$ 1, mas ela não será cobrada.

**Via Chat**

Você também pode verificar a sua conta através do nosso servidor Discord. Aqui, um moderador revisará sua conta e, se você for elegível, vai te aprovar para receber recompensas. Nosso discord [aqui](https://discord.com/invite/sWedwsNWjX).

Nessa etapa, as solicitações de verificação são analisadas manualmente, o que significa que podem levar de algumas horas a vários dias para serem aprovadas.

:::info Para acessar o nosso guia completo sobre recompensas e monetização na Odysee, clique [aqui](/pt/category-monetization/).

:::

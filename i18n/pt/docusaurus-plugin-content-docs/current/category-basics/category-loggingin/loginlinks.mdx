# Link mágico

Se preferir fazer login no Odysee usando um link enviado para o seu e-mail em vez de uma senha, você pode selecionar a opção **Usar Magic Link**.

![](https://i.imgur.com/sxATuIE.png)

Isso envia um e-mail de confirmação com um link. Basta clicar no botão ou no link do e-mail e preencher um CAPTCHA no navegador da Web ou no dispositivo e o login será feito com sucesso.

:::info Informação

Para melhorar a experiência de verificação, tente usar um navegador compatível e desative os escudos de proteção contra anúncios ou captcha. O link é válido apenas por alguns minutos, portanto, talvez seja necessário tentar novamente.

:::

![](https://i.imgur.com/MttvyYb.png)

:::danger PERIGO

Se você receber este e-mail com um link mágico, mas não estiver tentando fazer login na sua conta, não clique no link.

:::

## Aqui estão algumas etapas adicionais caso você encontre erros inesperados ao usar o link mágico.

1. feche seu navegador da Web (Firefox, Chrome, Brave).
2. Abra o Odysee e faça login usando o link mágico.
3. Clique no link enviado para seu e-mail no mesmo navegador.
4. Aguarde alguns segundos e sua sessão estará ativa.

:::info Informação

Em alguns casos, talvez seja necessário limpar o cache do navegador da Web para a Odysee se ocorrer um erro inesperado. Envie-nos um e-mail se tiver um erro inesperado em hello@odysee.com.

:::

## Por que estamos solicitado que você verifique via link?

Às vezes, ao fazer login, você pode receber um e-mail de confirmação sem solicitá-lo. Isso acontece quando detectamos uma mudança de IP. O motivo de segurança por trás disso é impedir que pessoas que não sejam você acessem sua conta, mesmo que saibam seu nome de usuário e senha.

Se você trocar de local entre logins, será solicitado que você confirme por e-mail, mesmo que tenha usado a Odysee em ambos os locais anteriormente.

Um e-mail de confirmação também pode ser enviado se você usar uma VPN.

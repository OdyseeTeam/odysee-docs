# Criação de lista de reprodução

Para criar uma nova playlist, primeiro clique no botão **Playlists** na barra lateral.

Nesse menu, clique em **Criar lista de reprodução**. Basta dar um nome e clicar na marca de seleção, e ele será criado.

Até serem publicadas, as listas de reprodução criadas não ficarão visíveis para ninguém, exceto para você.

## Adicionar conteúdo

Para adicionar um vídeo a uma lista de reprodução, você tem algumas opções.

Quando não estiver assistindo a um vídeo, clique no menu de três pontos, selecione **Adicionar à lista de reprodução** e selecione a lista de reprodução à qual deseja adicionar o vídeo.

Ao assistir a um vídeo, clique no botão **Salvar** abaixo do vídeo e selecione a playlist à qual deseja adicionar o vídeo.

# Assinaturas

Bem-vindo à seção de membros! Esta seção cobrirá tudo o que há para saber sobre o novo programa de associação da Odysee.

O que são associações? As assinaturas permitem que um criador crie níveis de assinatura para os fãs apoiarem com doações mensais. Os fãs podem doar para um criador por meio do canal ou do Portal do Doador.

Analisaremos as assinaturas do ponto de vista do criador e do membro.

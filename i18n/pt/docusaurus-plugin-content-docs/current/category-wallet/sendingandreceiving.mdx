# LBC enviando e recebendo

## Enviando

Para enviar LBC da carteira, clique no botão **Enviar** na tela da carteira.

A partir daqui, você pode inserir um endereço direto. Se você tiver uma carteira LBC fora do Odysee, é aqui que você a colocaria.

Como alternativa, você pode usar a guia Pesquisar para encontrar criadores no Odysee para enviar LBCs.

## Receber

Para receber LBC, clique no botão **Receber** na tela da carteira.

A partir daqui, você receberá o endereço da sua carteira. Se alguém enviar LBC para este endereço, ele aparecerá em sua carteira no Odysee.

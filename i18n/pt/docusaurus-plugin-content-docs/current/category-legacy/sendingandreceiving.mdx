# Créditos enviando e recebendo

## Enviando

Para enviar Créditos da carteira, clique no botão **Enviar** na tela da carteira.

A partir daqui, você pode inserir um endereço direto. Se você tiver uma carteira Créditos fora do Odysee, é aqui que você a colocaria.

Como alternativa, você pode usar a guia Pesquisar para encontrar criadores no Odysee para enviar Créditos.

## Receber

Para receber Créditos, clique no botão **Receber** na tela da carteira.

A partir daqui, você receberá o endereço da sua carteira. Se alguém enviar Créditos para este endereço, ele aparecerá em sua carteira no Odysee.

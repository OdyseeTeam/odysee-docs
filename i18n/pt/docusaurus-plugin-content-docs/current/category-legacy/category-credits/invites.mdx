# Convites

Você pode receber Créditos apenas convidando outras pessoas para o Odysee. No [odysee.com/$/invite](odysee.com/$/invite), você receberá um link de convite.

![](/img/en/category-monetization3.png)

Você também pode acessar esta página clicando no ícone do avatar no canto superior direito da tela e selecionando **Convites** no menu suspenso.

Compartilhe este link e, para cada usuário único que clicar nele, você receberá alguns Créditos.

Isso é especialmente útil para criadores que compartilham seu canal Odysee com seus espectadores. Em vez de fornecer o link básico do seu canal, forneça o link de convite.

Observe que um convite exigível tem a mesma definição de um usuário verificado do Odysee. Isso significa que o usuário indicado deve ter feito login no Odysee e verificado sua conta para receber alguns Créditos. Se você vir Não reivindicável, isso significa que eles não foram verificados ou não são elegíveis.

# Obtenha alguns Créditos

Para visualizar a página, clique no ícone do avatar no canto superior direito da tela e selecione a página no menu suspenso.

![](/img/en/category-monetization1.png)

O topo da tela mostrará a lista para obter mais Créditos. São eles:

- **Primeiro carregamento**: lance seu próprio canal compartilhando um vídeo.
- **Criador Cobiçado**: Este Crédito é concedido com base no número total de seguidores verificados (seguidores com contas verificadas) que seu canal possui.
- **A Jornada**: Este Crédito é concedido aos seguintes criadores.
- **Assistir semanalmente**: Este Crédito é concedido para assistir vídeos por pelo menos 3 dias em uma determinada semana.
- **Cinco Favoritos**: Este Crédito é concedido para republicar conteúdo de 5 criadores diferentes.
- **Código personalizado**: Se você recebeu um código personalizado de um criador, é aqui que você pode inseri-lo para obter Créditos.
- **Convites**: você pode receber Créditos simplesmente por convidar outras pessoas para o Odysee. Para obter mais informações, clique adiante para a próxima seção.

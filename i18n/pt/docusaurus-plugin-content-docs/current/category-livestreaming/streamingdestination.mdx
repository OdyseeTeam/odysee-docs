# Configurar um destino de stream

Depois que seu livestream for criado, mude para a aba **RTMP Setup**.

![odysee](https://thumbs.odycdn.com/6d8fc8bfc59ef6c442e3d0f545cf21e5.webp)

Seu servidor de stream e sua stream key são usados para informar ao seu software de transmissão para onde enviar o livestream. Você deve copiá-los e inseri-los onde for necessário no seu software de streaming.

## OBS

No OBS, clique em File > Settings > Stream.

![](/img/en/category-livestreaming5.png)

Em **Service**, selecione _Custom_.

Em **Server**, insira o servidor de stream fornecido pelo Odysee.

Em **Stream Key**, insira a stream key fornecida pelo Odysee.

Se tudo estiver correto, clique em **Ok**.

## Streamyard

:::info Nota:

Para fazer livestream para o Odysee usando Streamyard, você precisará ter a versão paga do Streamyard.

:::

Na tela principal do Streamyard, depois de fazer login, selecione **Destinations** no lado direito.

![](/img/en/category-livestreaming6.png)

A partir daí, selecione **Add Destination** e **Custom RTMP**.

Na tela seguinte, você começará a inserir as informações do seu stream.

Em **RTMP server URL**, insira o servidor de stream fornecido pelo Odysee.

Em **Stream Key**, insira a stream key fornecida pelo Odysee.

Se tudo estiver correto, clique em **Add RTMP Server**.

## Restream

:::info Nota:

Para fazer livestream para o Odysee usando Restream, você precisará ter a versão paga do Restream.

:::

![](/img/en/category-livestreaming7.png)

Na tela principal do Restream, depois de fazer login, selecione **Stream With Studio**.

![](/img/en/category-livestreaming8.png)

Quando estiver no studio, selecione **Add Destination** e **Custom RTMP**.

![](/img/en/category-livestreaming9.png)

A partir daí, você começará a inserir as informações do seu stream.

Em **RTMP URL**, insira o servidor de stream fornecido pelo Odysee.

Em **Stream Key**, insira a stream key fornecida pelo Odysee.

Se tudo estiver correto, clique em **Add Channel**.

:::info

É recomendável usar as configurações de saída no OBS sugeridas pelo Odysee. Caso sua conexão com a Internet e seu hardware permitam transmitir com qualidade, velocidade e resolução maiores, você pode testar outras configurações de saída.

:::

# Como faço para obter mais LBC?

## Recompensas

Através das recompensas LBC, você pode ganhar LBC apenas usando o Odysee.

A maioria das contas é aprovada para recompensas automaticamente durante o processo de inscrição.

Dependendo de fatores como localização, uso de VPN, etc., podem ser solicitadas informações adicionais para verificação. Para fazer isso, você tem algumas opções:

**Através do número de telefone**

Sua primeira opção é verificar através do seu telefone. Enviaremos uma mensagem de texto com um código.

**Através do Cartão**

Você também pode verificar usando um cartão de crédito ou débito. Não guardamos as informações do seu cartão nem lhe cobramos quaisquer taxas. Você pode ver uma autorização de US$ 1, mas ela cairá.

**Através do Chat**

Finalmente, você pode verificar através do nosso servidor Discord. Aqui, um moderador revisará sua conta e, se você for elegível, aprovará você para receber recompensas. Nossa Discord é [aqui](https://discord.com/invite/sWedwsNWjX).

Neste momento, as solicitações de verificação são analisadas manualmente, o que significa que podem levar de algumas horas a vários dias para serem aprovadas.

Para nosso guia completo sobre recompensas e monetização do Odysee, clique [aqui](https://help.odysee.tv/pt/category-monetization/).

## Convites

Você pode ganhar LBC convidando outras pessoas para o Odysee. Em odysee.com/$/invite, você receberá um link de convite.

Compartilhe este link e, para cada usuário único que clicar nele, você receberá um pouco de LBC.

Isso é especialmente útil para criadores que estão compartilhando seu canal Odysee com seus espectadores. Em vez de fornecer o link básico do seu canal, forneça o link de convite.

## Dicas LBC

As dicas de LBC dos espectadores também são uma maneira de ganhar mais LBC. Quando as dicas são enviadas pela primeira vez, elas aumentam o conteúdo sobre o qual foram informadas e não estarão disponíveis para uso até que sejam desbloqueadas. Para desbloquear uma dica, primeiro acesse a carteira clicando em **Sua Carteira**.

Nessa tela, você selecionará **Transações** e alterará o **Tipo** para Recebido.

Quando você vir o aumento que deseja reembolsar, basta clicar no ícone de desbloqueio ao lado de "Dica" e você desbloqueará a dica.

**Observação: uma vez que uma dica é desbloqueada, ela não irá mais impulsionar o conteúdo originalmente informado.**

# Denominação

Ao visualizar um canal ou conteúdo, você pode ter notado dois pontos seguidos por um número ou letra no final do URL.

![](https://i.imgur.com/USJ2H74.png)

Isso não é apenas para mostrar ou uma decisão estilística. Isso indica o *Claim ID* desse canal ou conteúdo específico, que é simplesmente um registro de seus metadados no LBRY.

Isso está relacionado ao sistema de apostas do LBRY. Por exemplo, suponha que haja dois uploads com o mesmo URL: "cat".

Um URL mostra: "odysee.com/cat:7b" e o outro mostra "odysee.com/cat:a"

Se você clicasse no link "odysee.com/cat", para qual desses dois vídeos, ambos com a URL "cat", o link iria?

A resposta é o vídeo com a maior participação na LBC. Digamos que "cat:7b" tenha a maior participação na LBC. Para encontrar este vídeo, você pode pesquisar por "odysee.com/cat:7b" ou simplesmente pesquisar por "odysee.com/cat" porque tem a maior aposta.

Para encontrar o vídeo com a aposta mais baixa, você precisa pesquisar o URL completo, "odysee.com/cat:a".

O conteúdo altamente licitado ou apostado também será refletido nos resultados de pesquisa no Odysee.

![](https://i.imgur.com/cvN25DQ.png)

Se você fizer uma pesquisa por "gato" no Odysee, o arquivo com a maior participação ou suporte LBC para esse resultado aparecerá na parte superior dos resultados.

Se a reivindicação mais alta fosse superada por uma aposta mais alta, o URL não mudaria imediatamente. Para cada mês que um nome é verificado, 1 dia é adicionado ao período de espera, até um máximo de 7 dias (após 7 meses).

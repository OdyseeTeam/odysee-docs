# Blockchain do Odysee

Um dos principais diferenciais do Odysee de outras plataformas de mídia é que o Odysee é construído em uma blockchain chamada LBRY.

Nesta seção, exploraremos o que exatamente isso significa e como isso afeta o Odysee. É importante notar que ninguém precisa saber nada sobre blockchains para usar e aproveitar o Odysee. É semelhante a dirigir um carro, você pode dirigir um sem entender como o motor funciona.

Esta seção se concentrará principalmente em como o Odysee interage com o protocolo LBRY. Para uma visão mais detalhada de todos os aspectos do protocolo LBRY, visite https://lbry.com/faq.

# Envie dicas LBC

Para enviar uma dica, clique no botão **Suporte** abaixo do conteúdo.

![](/img/es/image_42.png)

Esta opção também está disponível nos canais.

![](/img/es/image_43.png)

Nesse menu, você selecionará o botão **Dica** com o ícone LBC, selecionará um valor e clicará em **Enviar Dica**.

Esta dica da LBC aumentará o conteúdo/canal que você está apoiando, até que o criador decida desbloqueá-lo para uso em outro lugar no Odysee.

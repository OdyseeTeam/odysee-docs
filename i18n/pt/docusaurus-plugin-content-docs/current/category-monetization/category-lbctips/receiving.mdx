# Recebendo dicas LBC

Quando uma dica LBC é enviada enquanto assiste a um conteúdo, ela aumenta a participação LBC desse conteúdo até que seja desbloqueado. Até ser desbloqueado, não pode ser usado em nenhum outro lugar no Odysee.

Para desbloquear uma dica, primeiro vá para a carteira clicando <OdyseeButton url="https://odysee.com/$/wallet"><OdyseeIcon type="credits" /> Sua carteira</OdyseeButton>.

![](/img/es/image_1.png)

Nessa tela, você selecionará **Transações** e alterará o **Tipo** para Recebido.

![](/img/es/image_46.png)

Quando você vir o Boost que deseja reembolsar, basta clicar no ícone de desbloqueio ao lado de "Dica" e ele desbloqueará a dica.

:::info

Observação: uma vez que uma dica é desbloqueada, ela não aumentará mais o conteúdo em que foi originalmente suportada.

:::

Para obter mais informações sobre aprimoramentos de conteúdo e o sistema de staking Odysee, consulte nossa seção sobre staking clicando [aqui](/pt/category-blockchain/category-staking/naming).

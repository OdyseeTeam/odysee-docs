# Localização da Carteira

Ao usar o Odysee, você deve ter notado o ícone de um dólar e um valor numérico na barra superior. Isso mostra seu saldo total de AR. Clicar nele o levará para a página da carteira.

![Wallet](https://thumbs.odycdn.com/e6e0da12567980d225af59fdfb317f86.webp)

Após entrar na página da carteira, você verá a nova seção de carteira de AR. Para usar AR no Odysee, você precisa criar e/ou fazer login no Wander, uma carteira de criptomoedas compatível com AR. Agora você está pronto para criar sua carteira.

![Wallet](https://thumbs.odycdn.com/182939910a572c0aabb2ef39477fe4b0.webp)

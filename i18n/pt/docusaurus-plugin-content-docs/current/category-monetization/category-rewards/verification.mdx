# Verificar

A verificação permite que você ganhe créditos LBRY (LBC para abreviar) simplesmente usando o Odysee. Este é o nosso programa de recompensas. Exigimos verificação para ganhar recompensas e evitar abusos.

A maioria das contas é verificada automaticamente. Dependendo de fatores como localização, uso de VPN, etc., informações adicionais podem ser necessárias para verificação. Para fazer isso, você tem algumas opções:

**Através do número de telefone**

Sua primeira opção é verificar através do seu telefone. Enviaremos uma mensagem de texto com um código.

**Através do cartão**

Você também pode verificar usando um cartão de crédito ou débito. Não armazenaremos as informações do seu cartão nem cobraremos nenhuma taxa. Você pode ver uma autorização de US$ 1, mas ela voltará para o seu cartão.

**Através do bate-papo**

Finalmente, você pode verificar através do nosso servidor Discord. Aqui, um moderador revisará sua conta e, se for elegível, aprovará suas recompensas. Nosso Discord está aqui.

Nesse momento, as solicitações de verificação são revisadas manualmente, o que significa que podem levar de algumas horas a vários dias para serem aprovadas.

# Convite

Você pode ganhar LBC apenas convidando outras pessoas para o Odysee. Em odysee.com/$/invite, você receberá um link de convite.

Você também pode acessar esta página clicando no ícone de avatar no canto superior direito da tela e selecionando **Convites** no menu suspenso.

![](/img/es/image_47.png)

Compartilhe este link e, para cada usuário único que clicar nele, você receberá algum LBC.

Isso é especialmente útil para criadores que compartilham seu canal Odysee com seus espectadores. Em vez de fornecer o link básico do seu canal, forneça o link de convite.

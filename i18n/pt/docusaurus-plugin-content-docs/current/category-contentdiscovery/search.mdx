# Procurar

## Pesquisa básica

Para realizar uma pesquisa básica do Odysee, basta digitar um termo na barra de pesquisa e pressionar a tecla Enter.

![](/img/es/image_58.png)

O primeiro resultado será o conteúdo com a maior participação nesse URL de pesquisa. Para obter mais informações sobre esse processo, consulte nossa seção sobre staking clicando [aqui](/pt/category-blockchain/category-staking/naming).

O restante dos resultados são classificados por relevância.

Em **Filtro**, você tem alguns controles:

- **Tipo**: você pode optar por pesquisar apenas canais ou apenas conteúdo.
- **Data de upload**: mostra apenas o conteúdo do último dia, semana, mês ou ano.
- **Classificar por**: você pode classificar por relevância, mais recente primeiro ou mais antigo primeiro.

## Pesquisa de tags

As pesquisas de tags são diferentes das pesquisas normais.

Para realizar uma pesquisa de tag, primeiro digite um termo de pesquisa na barra de pesquisa. Antes de pesquisar, você pode notar um botão que diz **Procurar** no menu suspenso.

![](/img/es/image_59.png)

Esta opção "Explorar" não está lá apenas para mostrar, na verdade é um botão que fornece resultados de pesquisa completamente diferentes:

Quando você faz uma pesquisa "Explorar", todos os vídeos marcados com o termo que você digitou aparecerão. Assim como as principais categorias, você pode classificar por Novo, Tendência e Principais.

## Configurações de pesquisa de idioma

Ao pesquisar, você pode encontrar muito conteúdo em diferentes idiomas. Para alterar isso, primeiro vá para o menu Configurações. Isso pode ser encontrado clicando no ícone da foto do perfil no canto superior direito e selecionando **Configurações** no menu suspenso.

A partir daqui, você clicará na caixa de seleção para pesquisar apenas no idioma selecionado por padrão. Isso significa apenas que as pesquisas só encontrarão conteúdo em seu próprio idioma.

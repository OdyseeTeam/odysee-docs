# Sincronização do YouTube

Carregar centenas de vídeos antigos para uma nova plataforma pode ser uma tarefa assustadora. Felizmente, o Odysee possui uma ferramenta de sincronização do YouTube que permite que os criadores do YouTube espelhem automaticamente todo o conteúdo passado e futuro em seu canal Odysee.

Nesta postagem, abordaremos algumas noções básicas sobre a própria ferramenta de sincronização, forneceremos um tutorial e responderemos a algumas perguntas frequentes.

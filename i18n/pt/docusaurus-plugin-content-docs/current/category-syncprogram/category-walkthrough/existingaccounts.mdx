# Sincronização com uma conta existente

Para sincronizar um canal do YouTube com uma conta Odysee existente, primeiro clique no ícone de pessoa <OdyseeButtonRound ><OdyseeIcon type="person"/></OdyseeButtonRound> no canto superior direito (se você já tiver um canal, este ícone será a foto do perfil do seu canal) e clique em **Canais**.

![](/img/es/image_36.png)

Nesta tela, você selecionará **Sincronizar canal do YouTube**.

![](/img/es/image_37.png)

## Por que usar a ferramenta de sincronização?

A **ferramenta de sincronização para desktop** foi criada para creators que precisam de mais flexibilidade. Ela é a solução ideal quando a sincronização automática falha ou não atende aos requisitos necessários.

Com o [**Odysee Sync**](https://sync.odysee.tv), você pode executá-lo no seu PC (Windows/Mac/Linux) para automatizar o processo. É muito fácil de usar: com apenas alguns cliques, seu computador baixa o conteúdo do YouTube e o envia para o Odysee.

## Por que isso é importante?

- **Migração sem bloqueios:** Permite transferir conteúdo dos seus canais que podem estar inativos ou que simplesmente não atendem aos requisitos da sincronização automática.
- **Superação de limites:** Permite sincronizar conteúdos que estão fora dos limites padrão, como vídeos excepcionalmente longos.
- **Arquivo completo:** Garante que você possa levar todo o seu catálogo de vídeos para o Odysee. Use essa ferramenta para ter controle total sobre o seu processo de migração.

### Baixe aqui:

**Windows:** https://sync.odysee.tv/public/OdyseeSync.exe

**Mac/Linux:** https://sync.odysee.tv

**Observação:** Recomendamos configurar a opção **Cookies** com o **Firefox** se alguns vídeos falharem. Os motivos mais comuns são um grande volume de vídeos ou conteúdo com restrição de idade.

:::tip Importante

- **Cookies do navegador:** Para canais com muitos vídeos, talvez seja necessário usar a opção "Use browser cookies" (clique com o botão direito > preferences na barra de tarefas).
  - Verificamos que o **Firefox** funciona melhor no Windows.
  - Recomendamos baixar o Firefox, entrar no YouTube por ele e depois ativar essa opção.
  - Isso também funciona bem para vídeos com restrição de idade.
- **Nova tentativa:** Para tentar novamente qualquer falha, no momento você precisará fechar e reiniciar o aplicativo.
- **Novos usuários:** A ferramenta de sincronização criará um novo canal para você.
- **Canais já sincronizados:** Se você enviou conteúdo manualmente enquanto a sincronização estava desativada, o conteúdo **não** será deduplicado.

:::

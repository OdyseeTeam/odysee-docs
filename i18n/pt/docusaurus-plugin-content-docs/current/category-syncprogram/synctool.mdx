## Por que usar a ferramenta de sincronização?

A **Ferramenta de Sincronização para Desktop** foi projetada para criadores que precisam de mais flexibilidade. É a solução ideal quando a sincronização automática falha ou não atende aos requisitos para sincronização automática.

## Por que é importante?

- **Migração sem bloqueios:** Permite transferir conteúdo de seus canais que podem estar inativos ou que simplesmente não atendem aos requisitos para sincronização automática.
- **Superar limites:** Permite sincronizar conteúdo que não se enquadra nos limites padrão, como vídeos excepcionalmente longos.
- **Arquivo completo:** Garante que você possa trazer todo o seu catálogo de vídeos para o Odysee. Use esta ferramenta para assumir total controle do seu processo de migração.

Se você tiver muitos envios manuais, peça ajuda primeiro por e-mail. Além disso, se o seu canal do YouTube não for elegível para sincronização automática com o Odysee, envie-nos um e-mail para que possamos ajudá-lo a ativar a ferramenta de sincronização em **[hello@odysee.com](mailto:hello@odysee.com)**.

:::tip Importante

- **Cookies do Navegador:** Para canais com muitos vídeos, pode ser necessário usar a opção "Usar cookies do navegador" (clique com o botão direito > preferências na barra de tarefas).
  - Achamos que o **Firefox** funciona melhor no Windows.
  - Você deve baixar e fazer login no YouTube no Firefox e, em seguida, definir essa opção.
  - Isso também funciona bem para vídeos com restrição de idade.
- **Tentar novamente:** Para tentar novamente qualquer falha, você precisará sair e reiniciar o aplicativo neste momento.
- **Novos Usuários:** A ferramenta de sincronização criará um novo canal para você. Para sincronizar em um canal Odysee existente, entre em contato com [hello@odysee.com](mailto:hello@odysee.com).
- **Canais Sincronizados Existentes:** Se você enviou conteúdo manualmente enquanto a sincronização estava desativada, o conteúdo **não** será desduplicado.

:::

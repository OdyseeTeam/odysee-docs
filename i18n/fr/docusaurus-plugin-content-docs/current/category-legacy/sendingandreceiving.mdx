# Envoi et réception de Crédits

## Envoi

Pour envoyer des Crédits depuis votre porte-monnaie, cliquez sur le bouton **Envoyer** depuis l'écran du porte-monnaie.

![](/img/en/category-wallet5.png)

À partir de là, vous pouvez saisir une adresse directe. Si vous avez un porte-monnaie Crédits en dehors d'Odysee, c'est ici que vous le collerez.

![](/img/en/category-wallet6.png)

Vous pouvez également utiliser l'onglet Recherche pour trouver des créateurs dans Odysee à qui envoyer des Crédits.

## Réception

Pour recevoir des Crédits, vous devez cliquer sur le bouton **Recevoir** dans l'écran du portefeuille.

![](/img/en/category-wallet7.png)

À partir de là, vous obtiendrez l'adresse de votre porte-monnaie. Si quelqu'un envoie des Crédits à cette adresse, ils apparaîtront dans votre portefeuille sur Odysee.

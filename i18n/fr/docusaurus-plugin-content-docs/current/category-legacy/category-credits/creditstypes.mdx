# Obtenez des Crédits

Pour afficher la page, cliquez sur l'icône d'avatar dans le coin supérieur droit de l'écran et sélectionnez la page dans le menu déroulant.

![](/img/en/category-monetization1.png)

Le haut de l'écran affichera la liste pour obtenir plus de Crédits. Ce sont :

- **Premier chargement**: lancez votre propre chaîne en partageant une vidéo.
- **Créateur convoité**: ce Crédit est accordé en fonction du nombre total d'abonnés vérifiés (abonnés avec des comptes vérifiés) de votre chaîne.
- **The Journey**: ce Crédit est accordé aux créateurs suivants.
- **Montagne hebdomadaire**: ce Crédit est accordé pour regarder des vidéos pendant au moins 3 jours au cours d'une semaine donnée.
- **Cinq favoris**: ce Crédit est accordé pour republier le contenu de 5 créateurs différents.
- **Code personnalisé**: Si vous avez reçu un code personnalisé d'un créateur, c'est ici que vous pouvez le saisir pour obtenir des Crédits.
- **Invitations**: Vous pouvez recevoir des Crédit simplement en invitant d'autres personnes sur Odysee. Pour plus d’informations, cliquez pour passer à la section suivante.

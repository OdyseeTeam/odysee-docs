## Recherche

## Recherche de base

Pour effectuer une recherche de base dans Odysee, il suffit de taper un terme dans la barre de recherche et d'appuyer sur la touche Entrée.

*image*

Le premier résultat sera le contenu dont la valeur est la plus élevée pour cette URL de recherche. Pour plus d'informations sur ce processus, consultez notre section sur la valeur en jeu en cliquant [ici](https://help.odysee.tv/category-blockchain/category-staking/naming).

Le reste des résultats est trié par pertinence.

Sous **Filtre**, vous disposez de quelques contrôles :

- **Type** : Vous pouvez choisir de rechercher uniquement des chaînes ou uniquement du contenu.
- Date de téléchargement** : Affichez uniquement le contenu du dernier jour, de la dernière semaine, du dernier mois ou de la dernière année.
- **Tri par** : Vous pouvez trier par Pertinence, Plus récent d'abord, ou Plus ancien d'abord.

## Recherche par étiquette

Les recherches par étiquette sont différentes des recherches ordinaires.

Pour effectuer une recherche par étiquette, saisissez d'abord un terme de recherche dans la barre de recherche. Avant de lancer la recherche, vous pouvez remarquer un bouton qui dit **Explorer** dans le menu déroulant.

*image*

Cette option "Explorer" n'est pas seulement là pour faire joli, c'est en fait un bouton qui vous donne des résultats de recherche complètement différents :

Lorsque vous effectuez une recherche "Explorer", toutes les vidéos associées au terme que vous avez saisi apparaissent. Comme dans les catégories principales, vous pouvez trier par Nouveau, Tendance et Top.

## Paramètres de recherche par langue

Lorsque vous effectuez des recherches, vous pouvez trouver beaucoup de contenu dans différentes langues. Pour modifier ce paramètre, accédez d'abord au menu Paramètres. Pour ce faire, cliquez sur l'icône de la photo de profil dans le coin supérieur droit, puis sélectionnez **Paramètres** dans le menu déroulant.

À partir de là, vous pouvez cocher la case permettant d'effectuer des recherches uniquement dans la langue sélectionnée par défaut. Cela signifie simplement que les recherches ne porteront que sur le contenu dans votre propre langue.

## Envoyer des pourboires en espèces

## Ajout d'un mode de paiement

Avant d'envoyer un pourboire en espèces, vous devez ajouter un moyen de paiement. 

Pour ajouter une carte, vous devez d'abord ouvrir votre porte-monnaie. Votre porte-monnaie se trouve en haut à droite de l'écran, dans la barre de menu.

*image*

Sur la version mobile, vous le trouverez vers le bas du menu latéral.

*image*

À partir de là, vous sélectionnez **Méthodes de paiement**.

Sur l'écran Méthodes de paiement, il vous suffit de saisir votre nom complet, le numéro de votre carte, sa date d'expiration et son code CVC à 3 chiffres.

Si vous avez déjà ajouté une carte, elle apparaîtra sur l'écran Méthodes de paiement.

## Envoi de pourboires en espèces

Pour envoyer un pourboire, vous devez cliquer sur le bouton **soutien** sous un téléchargement.

*image*

Cette option est également disponible sur les chaînes.

*image*

Dans ce menu, vous sélectionnez le bouton **Pourboire** avec le symbole USD ou Euro, vous choisissez un montant et vous cliquez sur **Envoyer le pourboire**.

# Eligibilité

L'éligibilité à la rémunération à la séance dépend des conditions suivantes :

- Le contenu doit être votre propre contenu original et ne pas être dérivé des œuvres originales d'autres personnes.
- Le contenu et la chaîne doivent être étiquetés avec précision.
- Le compte doit avoir au moins une semaine d'existence.

Le type de contenu suivant n'est pas éligible pour le gain par vue :

- Violation de nos directives communautaires
- Œuvres protégées par le droit d'auteur
- Contenu illégal
- Contenu controversé (violence réelle, torture, abus, incitation à la haine) - contenu qui contient un ou plusieurs de ces éléments mais qui n'est pas un contenu d'actualité/éducatif
- Spam
- Rechargements ou œuvres dérivées par-dessus d'autres contenus publics/libres de droits d'auteur/copyright commons.
- lié à des parachutages/de l'argent gratuit

Si vous pensez que votre chaîne n'a pas été monétisée alors qu'elle devrait l'être, veuillez nous [envoyer un courriel](hello@odysee.com) avec un lien et une description de votre chaîne.

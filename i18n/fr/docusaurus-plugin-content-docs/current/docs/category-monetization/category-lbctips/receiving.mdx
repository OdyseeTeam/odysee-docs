# Recevoir des pourboires LBC

Lorsqu'un pourboire LBC est envoyé pendant la consultation d'un contenu, il augmente la valeur LBC de ce contenu jusqu'à ce qu'il soit débloqué. Tant qu'il n'est pas débloqué, il ne peut pas être utilisé ailleurs sur Odysee.

Pour débloquer un pourboire, allez d'abord dans le porte-monnaie en cliquant sur **Votre porte-monnaie**.

*image*

Sur cet écran, vous sélectionnerez **Transactions**, et changerez le **Type** en Reçu.

*image*

Lorsque vous voyez le montant que vous voulez rembourser, cliquez simplement sur l'icône de déverrouillage à côté de " Pourboire ", et vous déverrouillerez le pourboire.

**Attention : une fois qu'un pourboire est déverrouillé, il ne boostera plus le contenu pour lequel il a été donné à l'origine**.

Pour plus d'informations sur les boosts de contenu et le système de jalonnement d'Odysee, consultez notre section sur la valeur en cliquant [ici](https://help.odysee.tv/category-blockchain/category-staking/naming).

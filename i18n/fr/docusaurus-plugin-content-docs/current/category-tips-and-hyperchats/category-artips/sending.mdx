# Envoi d'astuces AR

Pour envoyer un conseil, vous devez cliquer sur le bouton **Support** dans la bannière de la chaîne. Le bouton d'assistance se trouve également sous la vidéo.

![Conseils](https://thumbs.odycdn.com/bebfbd4cf6da73a7d4e691cbe4e9942b.webp)

## Cette option est également disponible sur le contenu.

Dans ce menu, sélectionnez le bouton **Pourboire** avec l'icône représentant un dollar, choisissez un montant et cliquez sur **Envoyer un pourboire**.

![Tips](https://thumbs.odycdn.com/a5b8e490d797ba077e3ffa0a449e499c.webp)

# Recevoir des conseils sur AR

En recevant un pourboire sur AR, vous bénéficiez du soutien de la communauté pour le contenu de votre chaîne. Ce pourboire sera transféré directement dans votre portefeuille Odysee. Vous pouvez consulter tous les pourboires reçus sur la page de votre portefeuille.

![wallet](https://thumbs.odycdn.com/e6e0da12567980d225af59fdfb317f86.webp)

À partir de cet écran, sélectionnez **Balance** et passez à l'onglet **Pourboire**. Vous y verrez tous les pourboires que vous avez reçus.

![Tips](https://thumbs.odycdn.com/04649ce7117cffdf0fcc6a6859401177.webp)

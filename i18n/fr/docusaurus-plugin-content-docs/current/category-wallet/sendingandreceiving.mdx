# Envoi et réception de LBC

## Envoi

Pour envoyer des LBC depuis votre porte-monnaie, cliquez sur le bouton **Envoyer** depuis l'écran du porte-monnaie.

À partir de là, vous pouvez saisir une adresse directe. Si vous avez un porte-monnaie LBC en dehors d'Odysee, c'est ici que vous le collerez.

Vous pouvez également utiliser l'onglet Recherche pour trouver des créateurs dans Odysee à qui envoyer des LBC.

## Réception

Pour recevoir des LBC, vous devez cliquer sur le bouton **Recevoir** dans l'écran du portefeuille.

À partir de là, vous obtiendrez l'adresse de votre porte-monnaie. Si quelqu'un envoie des LBC à cette adresse, ils apparaîtront dans votre portefeuille sur Odysee.

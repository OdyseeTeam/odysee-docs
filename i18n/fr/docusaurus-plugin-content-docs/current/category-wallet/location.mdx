# Emplacement du porte-monnaie

Lorsque vous êtes sur Odysee, vous avez peut-être remarqué l'icône LBC <OdyseeIcon type="credits"/> et une valeur numérique sur la barre supérieure.

<Image src="/img/fr/Ajouteruncomptebancaire01.png" />

Cela indique votre nombre total de crédits LBRY, et en cliquant dessus, vous accéderez à la page du portefeuille.

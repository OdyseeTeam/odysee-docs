Vous pouvez accéder au portail des donateurs dans la page d'accueil des *Adhésions*. Cliquez sur l'icône de l'avatar dans le coin supérieur droit et cliquez sur *Adhésions*.

![](https://i.imgur.com/MwllRtm.jpg)

Après avoir accédé à la page d'acceuil des *Adhésions*, vous verrez deux portails:

![](https://i.imgur.com/1IBQMGD.png)

Pour gérer vos adhésions, cliquez sur le bouton *Entrer dans le portail des donateurs*.

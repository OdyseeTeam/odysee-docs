# Vue d'ensemble

À partir de la page d'acceuil du portail des donateurs, vous pouvez afficher chacune des souscriptions aux chaînes que vous avez rejointes.

![](https://i.imgur.com/LqrcBBI.png)

Cela montre le nom de la chaîne que vous avez rejointe, le nom du niveau, depuis combien de temps vous êtes membre, le montant mensuel que vous avez rejoint et si l'adhésion est toujours active.

Le bouton rose avec l'icône de fusée vous amènera à la page d'adhésion de cette chaîne.

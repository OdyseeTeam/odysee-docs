# Rejoindre un abonnement

:::info

*Si vous n'avez pas ajouté de mode de paiement, vous ne pourrez pas devenir membre. Pour ce faire, vous pouvez consulter notre guide [ici](/category-basics/paymentmethod).*

:::

Pour rejoindre les niveaux d'adhésion d'une chaîne, vous pouvez soit sélectionner le bouton orange *Rejoindre* en haut de la page de sa chaîne, soit choisir un niveau dans l'onglet *Adhésion*.

![](https://i.imgur.com/llE6uFG.png)

Après avoir cliqué sur la sélection du bouton orange *Rejoindre*, vous pourrez sélectionner le niveau spécifique que vous souhaitez rejoindre.

![](https://i.imgur.com/2LKpAhp.png)

Une fois que vous avez sélectionné votre niveau, vous verrez le niveau que vous rejoignez, sa description, ses avantages et le montant de votre don mensuel.

![](https://i.imgur.com/5sDCYvH.png)

Si tout semble correct, appuyez sur confirmer. Vous êtes maintenant membre de la chaîne!

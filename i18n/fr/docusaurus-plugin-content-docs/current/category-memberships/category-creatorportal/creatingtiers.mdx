# Créer des niveaux d'adhésion

La première étape consiste à choisir la chaîne avec lequel vous souhaitez interagir. Par la suite, il vous sera demandé de configurer vos informations de compte bancaire, si vous ne l'avez pas déjà fait. Après les informations de compte bancaire, vous pouvez configurer et publier des niveaux de don.

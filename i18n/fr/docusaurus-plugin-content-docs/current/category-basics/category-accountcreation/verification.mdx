# Vérification
 Après avoir créé votre compte, il peut vous être demandé de vérifier votre identité. Si vous n'avez pas été invité à le faire, vous pouvez passer à la section suivante en cliquant sur [ici](link.com). 

La vérification n'est pas obligatoire pour utiliser Odysee, mais elle est fortement recommandée. La vérification vous permet de gagner des crédits LBRY (LBC en abrégé) simplement en utilisant Odysee. Nous exigeons une vérification pour gagner des LBC afin d'éviter les abus.

:::info
Vous voulez en savoir plus sur les LBC et leur rôle sur Odysee, consultez notre guide [ici](http://chihuahuaspin.com).

:::

La plupart des comptes sont vérifiés automatiquement. En fonction de facteurs tels que la localisation, l'utilisation d'un VPN, etc., des informations supplémentaires peuvent vous être demandées pour la vérification. Pour ce faire, vous avez plusieurs possibilités :

<Image src="/img/fr/Vérification01.png" />

**Par numéro de téléphone**

Votre première option est de vérifier par téléphone. Nous vous enverrons un SMS avec un code.

**Par carte** 

Vous pouvez également vérifier votre identité en utilisant une carte de crédit ou de débit. Nous n'enregistrerons pas les informations de votre carte et ne vous facturerons aucun frais. Il se peut que vous constatiez une autorisation de 1 $, mais celle-ci disparaîtra.

**Via Chat**

Enfin, vous pouvez vérifier par le biais de notre serveur Discord. Ici, un modérateur examinera votre compte et, si vous êtes admissible, vous approuvera pour les récompenses. Notre Discord est [ici](https://discord.com/invite/sWedwsNWjX).

Pour le moment, les demandes de vérification sont examinées manuellement, ce qui signifie que leur approbation peut prendre de quelques heures à plusieurs jours.

:::info 
Pour notre guide complet sur les récompenses et la monétisation d'Odysee, cliquez [ici](http://chihuahuaspin.com).

:::

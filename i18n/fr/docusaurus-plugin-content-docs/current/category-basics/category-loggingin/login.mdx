# Se connecter à Odysee

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs>
  <TabItem value="web" label="Web" default>
    Vous avez déjà un compte et vous revenez sur Odysee ? Vous cliquerez sur le bouton <a href="https://odysee.com/$/signin"><b>Se connecter</b></a> dans le coin supérieur droit.
    <Image img="/img/fr/SeconnecteràOdysee01.png" />
  </TabItem>
  <TabItem value="mobile" label="Mobile">
    Sur un mobile, vous cliquerez sur l'icône de la Personensymbol <OdyseeButtonRound ><OdyseeIcon type="person"/></OdyseeButtonRound>, ce qui affichera Connexion dans le menu déroulant <a href="https://odysee.com/$/signin"><b>Anmelden</b></a>.
    <Image img="/img/fr/Mseconnecter.png" />
  </TabItem>
</Tabs>

Vous commencerez par saisir votre adresse électronique.

<Image src="/img/fr/SeconnecteràOdysee02.png" />

Sur l'écran du mot de passe, vous avez quelques options.

<Image src="/img/fr/SeconnecteràOdysee03b.png" />

Si vous connaissez votre mot de passe et souhaitez vous connecter, entrez-le simplement et cliquez sur Suivant . Alternativement, vous pouvez également faire envoyer un [lien magique](/category-basics/category-loggingin/loginlinks) à votre adresse e-mail ou réinitialiser votre [mot de passe](/category-basics/category-loggingin/passwordreset). .

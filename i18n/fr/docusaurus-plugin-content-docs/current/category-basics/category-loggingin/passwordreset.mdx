# Réinitialiser votre mot de passe

Si vous ne connaissez pas votre mot de passe, votre première possibilité est [Page de connexion](https://odysee.com/$/signin) et de cliquer sur ** Mot de passe oublié ? **

À partir de là, vous devrez saisir à nouveau votre adresse électronique, cliquer sur **Réinitialiser le mot de passe**, et nous vous enverrons un courrier électronique pour réinitialiser votre mot de passe.

<Image src="/img/fr/Réinitialiservotremotdepasse01.png" />

Suivez le lien figurant dans le courriel pour définir un nouveau mot de passe.

<Image src="/img/fr/Réinitialiservotremotdepasse02.png" />

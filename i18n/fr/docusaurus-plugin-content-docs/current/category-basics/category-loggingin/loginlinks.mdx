# Lien magique

Si vous préférez vous connecter à Odysee en utilisant un lien envoyé à votre e-mail au lieu d'un mot de passe, vous pouvez sélectionner l'option **Utiliser le lien magique**.

<Image src="/img/fr/Liendeconnexion.png" />

Un courriel de confirmation contenant un lien est alors envoyé. Il vous suffit de cliquer sur le bouton ou le lien de l'e-mail et de remplir un CAPTCHA dans votre navigateur web ou votre appareil, et vous serez connecté avec succès.

:::info Informations

Pour améliorer l'expérience de vérification, essayez d'utiliser un navigateur web compatible et désactivez les boucliers de protection contre les publicités ou les captchas. Le lien n'est valable que quelques minutes, il se peut donc que vous deviez réessayer.

:::

<Image src="https://i.imgur.com/MttvyYb.png" />

:::danger Attention

Si vous recevez ce lien magique par courrier électronique mais que vous n'essayez pas de vous connecter à votre compte, ne cliquez pas sur le lien.

:::

## Voici quelques étapes supplémentaires au cas où vous rencontreriez des erreurs inattendues lors de l'utilisation du lien magique.

1. Fermez votre navigateur web (Firefox, Chrome, Brave).
2. Ouvrez Odysee et connectez-vous en utilisant le lien magique.
3. Cliquez sur le lien envoyé à votre email dans le même navigateur.
4. Attendez quelques secondes et votre session sera active.

:::info Informations

Dans certains cas, il peut être nécessaire de vider le cache de votre navigateur Internet pour Odysee si vous rencontrez des erreurs inattendues. Envoyez-nous un courriel si vous rencontrez une erreur inattendue à l'adresse hello@odysee.com.

:::

## Pourquoi me demande-t-on de vérifier par un lien?

Parfois, lorsque vous vous connectez, il se peut que vous receviez un courriel de confirmation sans l'avoir demandé. Cela se produit lorsque nous détectons un changement d'IP. La raison de sécurité derrière cela est d'empêcher les personnes autre que vous de se connecter à votre compte, même si elles connaissent votre nom d'utilisateur et votre mot de passe.

Si vous changez de lieu entre deux connexions, il vous sera demandé de confirmer par courriel, même si vous avez déjà utilisé Odysee à ces deux endroits.

Cela peut également être enclenché si vous utilisez un VPN.

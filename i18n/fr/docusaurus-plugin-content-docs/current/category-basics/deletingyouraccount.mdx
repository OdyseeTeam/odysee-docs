# Suppression de votre compte

Actuellement, la suppression du compte est en partie un processus manuel. Vous pouvez supprimer tout le contenu de votre compte et envoyer une demande de suppression de compte en utilisant l'option «Supprimer le compte» dans les [Paramètres](https://odysee.com/$/settings). Un e-mail de confirmation sera envoyé une fois la suppression traitée.
![](https://thumbs.odycdn.com/4d8fc2235830ac249751daa82c42bffc.webp)

:::info

Si l'option n'est pas disponible sur l'application que vous utilisez, veuillez d'abord supprimer toutes les chaînes et tous les contenus téléchargés du compte, puis envoyez un courriel à hello@odysee.com pour demander la suppression du compte.

:::

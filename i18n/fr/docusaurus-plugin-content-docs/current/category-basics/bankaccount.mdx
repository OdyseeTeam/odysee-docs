# Ajouter un compte bancaire

En ajoutant un compte bancaire, vous serez en mesure de recevoir des pourboires en espèces de la part d'autres utilisateurs. Nous encourageons vivement les créateurs à le faire, afin que leurs spectateurs puissent leur envoyer des pourboires.

Pour ajouter un compte bancaire, vous devez d'abord ouvrir <OdyseeButton url="https://odysee.com/$/wallet"><OdyseeIcon type="credits" /> Porte-monnaie</OdyseeButton>. Votre porte-monnaie se trouve en haut à droite de l'écran, dans la barre de menu.

<Image src="/img/fr/Ajouteruncomptebancaire01.png" />

Sur la version mobile, vous le trouverez en bas du menu latéral.

<Image src="/img/fr/Mportefeuille.png" />

De là, vous sélectionnerez **Comptes bancaires**.

<Image src="/img/fr/Ajouteruncomptebancaire02.png" />

Cela vous amènera ici, où vous sélectionnerez **Connecter votre compte bancaire**.

<Image src="/img/fr/Ajouteruncomptebancaire03.png" />

Cela vous permet de sortir d'Odysee pour aller sur Stripe.

<Image src="/img/fr/Ajouteruncomptebancaire04.png" />

À partir de là, il vous suffit de saisir les informations demandées.

Une fois que votre compte bancaire est ajouté et que vous recevez des pourboires, vous pourrez suivre le montant reçu sur votre portefeuille.

<Image src="/img/fr/Ajouteruncomptebancaire05.png" />

:::info
  Tous les 2 ou 3 jours ouvrables, les pourboires envoyés sur votre porte-monnaie Odysee seront automatiquement encaissés et envoyés sur votre compte bancaire.

:::

# Ajouter une méthode de paiement

En ajoutant une carte de crédit ou de débit, vous pourrez donner des pourboires en espèces aux créateurs, précommander des vidéos, et plus encore.

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs groupId="device">
  <TabItem value="web" label="Web">
    Pour ajouter une carte, vous devez d'abord ouvrir votre <OdyseeButton url="https://odysee.com/$/wallet"><OdyseeIcon type="credits" />  porte-monnaie</OdyseeButton>. Ce dernier se trouve en haut à droite de l'écran, dans la barre de menu.
    <Image img="/img/fr/Ajouteruneméthodedepaiement01.png" />
  </TabItem>
  <TabItem value="mobile" label="Mobile">
    Pour ajouter une carte, vous devez d'abord ouvrir votre <OdyseeButton url="https://odysee.com/$/wallet"><OdyseeIcon type="credits" />  porte-monnaie</OdyseeButton>.Sur la version mobile, vous le trouverez en bas du menu latéral.
    <Image img="/img/fr/Mportefeuille.png" />
  </TabItem>
</Tabs>

De là, vous sélectionnerez **Méthodes de paiement**.

<Image src="/img/fr/Ajouteruneméthodedepaiement02.png" />

Sur l'écran "[Méthodes de paiement](https://odysee.com/$/settings/card)", il vous suffit de saisir votre nom complet, le numéro de votre carte, sa date d'expiration et son code CVC à 3 chiffres.

<Image src="/img/fr/paiement.png" />

Si vous avez déjà ajouté une carte, elle apparaîtra sur l'écran "[Méthodes de paiement](https://odysee.com/$/settings/card)".

![](https://i.imgur.com/pUuzmPC.png)

À partir de là, vous pouvez **supprimer la carte**, **consulter l'historique de vos transactions**, et **changer votre devise de USD en Euros, ou vice versa**.

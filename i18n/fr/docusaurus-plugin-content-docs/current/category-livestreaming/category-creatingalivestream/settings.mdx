# Paramètres et personnalisation

Si c'est votre première diffusion en direct sur Odysee, vous devrez créer un livestream.

## Titre, URL, description

![odysee](https://thumbs.odycdn.com/6c9c0b8cd69647077d5b08dc36f28492.webp)

Le premier champ obligatoire est le **Titre**. Nous recommandons des titres qui informent clairement les utilisateurs du contenu, tout en étant suffisamment attractifs pour leur donner envie de cliquer.

Ensuite vient l'**URL** du fichier. Elle est distincte du titre. Le titre est ce qui s'affiche sur la publication, tandis que l'URL correspond simplement à ce qui apparaît dans l'adresse de la vidéo.

Après l'URL, nous vous demandons une **Description**. Le bourrage de mots-clés dans la description n'a aucun effet sur la recherche, nous vous recommandons donc d'utiliser simplement ce champ pour expliquer votre vidéo avec autant ou aussi peu de mots que nécessaire.

## Date du stream

Sur Odysee Livestreams, il existe deux options de date : **Anytime** et **Scheduled Time**.

![odysee](https://thumbs.odycdn.com/7ea37a035ecc9d0ea5f983f95dcc6063.webp)

Lorsque la date est définie sur **Scheduled Time**, votre livestream apparaîtra à l'avance sur votre chaîne et dans les flux de vos abonnés, avec l'heure programmée affichée. Cela permet à vos abonnés de savoir à l'avance que vous prévoyez de diffuser.

Lorsque la date est définie sur **Anytime**, votre livestream n'apparaîtra pas tant que vous n'aurez pas commencé à diffuser.

## Vignette et balises

![odysee](https://thumbs.odycdn.com/54ebb104d47902ff4d5800e3b671473d.webp)

Après la date, il vous sera demandé d'ajouter une **Vignette**.

Pour les vignettes, nous recommandons 1280 x 720 pixels. Vous pouvez téléverser un fichier ou coller un lien d'image.

Ensuite, il vous sera demandé d'ajouter des **Balises**.

![odysee](https://thumbs.odycdn.com/424e15f69fd3b030698006b24af3f162.webp)

Les balises améliorent la découvrabilité du contenu. Nous recommandons des balises simples en lien avec le genre de votre contenu, par exemple Art, Gaming, Comédie, etc.

À l'heure actuelle, chaque livestream ne peut avoir que 5 balises maximum.

## Interactions utilisateur

![odysee](https://thumbs.odycdn.com/100ba10f1e4f2a882628d7c0fdad8976.webp)

Les créateurs et les propriétaires de chaînes peuvent désactiver les réactions (fire et slime) depuis le formulaire de publication dans les options **Controls Tags**. Vous pouvez désactiver les réactions pour les **commentaires** ou pour **Tipping and Boosting**.

Cette fonctionnalité est facultative et n'est pas activée par défaut. Les propriétaires de chaînes doivent activer les contrôles de balises depuis le formulaire de publication, lors de l'édition du contenu ou depuis les paramètres de la chaîne.

## Paramètres supplémentaires

![odysee](https://thumbs.odycdn.com/e555970bc07931d432528f5d039dbe1b.webp)

Sous cet onglet, plusieurs options sont disponibles.

La première option est **Language**. Par défaut, elle reprend votre langue sélectionnée.

L'option suivante est **License**. Elle sert à préciser la licence de droits d'auteur.

Par défaut, elle est réglée sur "None". Si aucune licence n'est sélectionnée, le fichier sera protégé par le droit d'auteur par défaut. Les autres licences disponibles sont :

- Les licences Public Domain autorisent la réutilisation sans restriction.
- Les licences Creative Commons autorisent la réutilisation avec attribution au créateur d'origine.
- Vous pouvez aussi définir la licence simplement comme Copyrighted.

## Sélection de la chaîne

La dernière option est la **Sélection de la chaîne**. Si vous avez plusieurs chaînes, c'est ici que vous choisirez sur laquelle le livestream sera hébergé.

![](/img/en/category-uploading8.png)

Si tous les paramètres sont corrects, cliquez sur **Upload**, puis sur **Confirm**.

La confirmation complète de votre livestream peut prendre quelques minutes, mais cela marque la fin du processus de création.

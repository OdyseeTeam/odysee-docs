# Configurer une destination de stream

Une fois votre livestream créé, passez à l'onglet **RTMP Setup**.

![odysee](https://thumbs.odycdn.com/6d8fc8bfc59ef6c442e3d0f545cf21e5.webp)

Votre serveur de stream et votre clé de stream permettent d'indiquer à votre logiciel de diffusion où envoyer votre livestream. Vous devrez les copier puis les renseigner aux endroits appropriés dans votre logiciel de streaming.

## OBS

Dans OBS, cliquez sur File > Settings > Stream.

![](/img/en/category-livestreaming5.png)

Dans **Service**, sélectionnez _Custom_.

Dans **Server**, saisissez le serveur de stream fourni par Odysee.

Dans **Stream Key**, saisissez la clé de stream fournie par Odysee.

Si tout est correct, cliquez sur **Ok**.

## Streamyard

:::info Note:

Pour diffuser en direct sur Odysee avec Streamyard, vous devez disposer de la version payante de Streamyard.

:::

Depuis l'écran principal de Streamyard après connexion, sélectionnez **Destinations** sur le côté droit.

![](/img/en/category-livestreaming6.png)

À partir de là, sélectionnez **Add Destination** puis **Custom RTMP**.

Sur l'écran suivant, commencez à renseigner les informations de votre stream.

Dans **RTMP server URL**, saisissez le serveur de stream fourni par Odysee.

Dans **Stream Key**, saisissez la clé de stream fournie par Odysee.

Si tout est correct, cliquez sur **Add RTMP Server**.

## Restream

:::info Note:

Pour diffuser en direct sur Odysee avec Restream, vous devez disposer de la version payante de Restream.

:::

![](/img/en/category-livestreaming7.png)

Depuis l'écran principal de Restream après connexion, sélectionnez **Stream With Studio**.

![](/img/en/category-livestreaming8.png)

Une fois dans le studio, sélectionnez **Add Destination** puis **Custom RTMP**.

![](/img/en/category-livestreaming9.png)

À partir de là, commencez à renseigner les informations de votre stream.

Dans **RTMP URL**, saisissez le serveur de stream fourni par Odysee.

Dans **Stream Key**, saisissez la clé de stream fournie par Odysee.

Si tout est correct, cliquez sur **Add Channel**.

:::info

Il est recommandé d'utiliser les paramètres de sortie dans OBS suggérés par Odysee. Si votre connexion Internet et votre matériel vous permettent de diffuser avec une qualité, une vitesse et une résolution supérieures, vous pouvez essayer d'autres paramètres de sortie.

:::

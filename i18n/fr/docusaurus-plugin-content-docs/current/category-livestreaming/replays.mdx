# Téléchargement des rediffusions

Pour l'instant, il n'existe pas d'option permettant de publier automatiquement les reprises de livestream.

Pour publier un ancien livestream, vous devez retourner à l'écran **Publier** sous l'onglet *Go Live*.

<Image src="/img/fr/Livereplay.png" />

À partir de là, vous sélectionnerez **Choose replay**. Si aucune rediffusion n'apparaît, vous pouvez l'actualiser en sélectionnant **Vérifier les rediffusions**.

Dans le menu, vous sélectionnerez la rediffusion du livestream que vous souhaitez publier, vous entrerez les informations demandées et vous sélectionnerez <PrimaryOdyseeButton>Créer</PrimaryOdyseeButton> en bas de la page.

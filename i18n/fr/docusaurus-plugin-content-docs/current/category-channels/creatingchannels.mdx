# Créer plus de chaînes

Pour créer une autre chaîne, vous devez d'abord retourner à la page **[<OdyseeIcon type="atsign"/> Chaînes](https://odysee.com/$/channels)**.

<Image src="/img/fr/Synchronisationcompteexistant01.png" />

À partir de cette page **[<OdyseeIcon type="atsign"/> Chaînes](https://odysee.com/$/channels)**, vous pourrez créer d'autres chaînes en utilisant le bouton **Nouvelle chaîne**.

<Image src="/img/fr/Créerdenouvelleschaines.png" />

:::info Bon à savoir
Vous pouvez avoir jusqu'à 5 chaînes.

:::

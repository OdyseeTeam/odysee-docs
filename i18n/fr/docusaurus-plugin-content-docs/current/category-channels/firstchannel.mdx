# Créer votre première chaîne

Pour pouvoir faire des choses comme aimer, ne pas aimer, commenter et créer des listes de lecture, vous devez créer une chaîne.

Cliquez sur l'icône représentant une personne dans le coin supérieur droit (si vous avez déjà une chaîne, cette icône sera l'image de profil de votre chaîne), puis cliquez sur **Chaînes**.

<Image src="/img/fr/Synchronisationcompteexistant01.png" />

De là, vous cliquerez sur **[Nouvelle chaîne](https://odysee.com/$/channel/new)**.

<Image src="/img/fr/Créervotrepremièrechaîne02.png" />

:::info
Si vous essayiez de synchroniser votre chaîne YouTube, vous pouvez également cliquer sur l'option **Synchroniser la chaîne YouTube** sans avoir à créer un nouveau compte.

:::

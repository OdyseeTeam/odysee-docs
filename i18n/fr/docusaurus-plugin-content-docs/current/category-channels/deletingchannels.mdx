# Suppression de chaînes

Pour supprimer une chaine, vous avez quelques options.

Vous pouvez les supprimer sur la page **[<OdyseeIcon type="atsign"/> Chaînes](https://odysee.com/$/channels)**, en utilisant le menu à trois boutons.

<Image src="/img/fr/Suppressiondechaînes01.png" />

Vous pouvez également supprimer une chaîne en utilisant le menu à trois boutons sur la chaîne elle-même. 

<Image src="/img/fr/Suppressiondechaînes02.png" />

Si vous décidez de supprimer une chaîne en cours d'édition, vous disposez également d'une option de suppression dans le menu d'édition.

<Image src="/img/fr/Suppressiondechaînes03A.png" />

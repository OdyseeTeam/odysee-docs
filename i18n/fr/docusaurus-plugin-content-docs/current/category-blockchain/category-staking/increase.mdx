# Les moyens d'augmenter la valeur de Crédits

La valeur de Crédits est en partie déterminée par les créateurs eux-mêmes et en partie déterminée par la communauté.

## Dépôts

La première façon d'augmenter la valeur d'un numéro de réclamation est d'effectuer un dépôt de crédit.

Lors du téléchargement initial d'un fichier, vous trouverez **Dépôt** sous l'onglet **Options supplémentaires**.

<Image src="/img/fr/augmentervaleurlbc1.png" />

Cela vous montrera votre valeur actuelle de mise (qui est par défaut de 0,01 Crédits), vous indiquera le Crédits nécessaire pour avoir la valeur la plus élevée de l'URL, et vous pourrez ajuster votre valeur.

Vous pouvez également ajuster le montant du dépôt après qu'un fichier ait été téléchargé. Il suffit de cliquer sur **Editer**, et dans le menu d'édition, vous pourrez ajuster le dépôt Crédits.

Sur les chaînes, vous pourrez modifier la valeur de la mise sous l'onglet **Détails du crédit** pendant l'édition.

## Boosts

Si vous souhaitez booster la valeur Crédits de quelqu'un d'autre, votre première option est le boosting.

Sous un chargement, cliquez sur le bouton **Soutien**.

<Image src="/img/fr/Soutien1.png" />

Cette option est également disponible sur les chaînes.

<Image src="/img/fr/soutien2.png" />

Dans ce menu, vous sélectionnerez **Booster**, choisirez un montant et cliquerez sur **Booster ce contenu**.

Les boosts sont entièrement remboursables. Jusqu'à ce qu'ils soient remboursés, ils augmenteront la valeur de ce contenu ou de cette chaîne.

Pour rembourser un boost, allez d'abord dans le porte-monnaie en cliquant sur votre <PrimaryOdyseeButton url="https://odysee.com/$/wallet"><OdyseeIcon type="credits"/>porte-monnaie</PrimaryOdyseeButton>.

<Image src="/img/fr/Ajouteruncomptebancaire01.png" />

Dans l'écran suivant, vous allez sélectionner **Transactions**, et changer le **Type** en Soutien.

<Image src="/img/fr/augmentervaleurlbc2.png" />

Lorsque vous voyez le boost que vous voulez vous faire rembourser, cliquez simplement sur la poubelle à côté de " Soutien ", et vous serez remboursé.

<Image src="/img/fr/augmentervaleurlbc3.png" />

## Pourboires Crédits

Lorsqu'un pourboire Crédits est envoyé pendant la consultation d'un contenu, il augmente la mise Crédits de ce contenu jusqu'à ce qu'il soit débloqué. Contrairement aux boosts, les **pourboires ne sont pas remboursables pour la personne qui les donne**.

Pour envoyer un pourboire, vous devez cliquer sur le bouton **Soutien** sous un téléchargement.

<Image src="/img/fr/Soutien1.png" />

Cette option est également disponible sur les chaînes.

<Image src="/img/fr/soutien2.png" />

Dans ce menu, vous devez sélectionner le bouton **Pourboire** avec l'icône Crédits, choisir un montant et cliquer sur **Envoyer le pourboire**.

Ce pourboire Crédits boostera le contenu/la chaîne sur lequel/laquelle il a été envoyé, jusqu'à ce que le créateur choisisse de le débloquer pour l'utiliser ailleurs sur Odysee.

Pour déverrouiller un pourboire, allez d'abord dans votre portefeuille en cliquant sur votre <PrimaryOdyseeButton url="https://odysee.com/$/wallet"><OdyseeIcon type="credits"/>porte-monnaie</PrimaryOdyseeButton>.

<Image src="/img/fr/Ajouteruncomptebancaire01.png" />

Dans cet écran, vous sélectionnerez **Transactions**, et changerez le **Type** en Reçu.

<Image src="/img/fr/ObtenirplusdeLBC1.png" />

Lorsque vous voyez le boost que vous souhaitez vous faire rembourser, cliquez simplement sur l'icône de déverrouillage à côté de "pourboire", et vous déverrouillerez le pourboire.

<Image src="/img/fr/ObtenirplusdeLBC2.png" />

:::caution Attention 
Une fois qu'un pourboire est déverrouillé, il ne boostera plus le contenu pour lequel il a été donné à l'origine.**
:::

# Introduction

L'un des principaux atouts d'Odysee par rapport aux autres plateformes médiatiques est qu'il est construit sur une blockchain appelée LBRY.

Dans cette section, nous allons explorer ce que cela signifie exactement et comment cela affecte Odysee. Il est important de noter que personne n'a besoin de connaître les blockchains pour utiliser et apprécier Odysee. C'est un peu comme conduire une voiture, vous pouvez en conduire une sans comprendre comment le moteur fonctionne.

Cette section se concentrera principalement sur la façon dont Odysee interagit avec le protocole LBRY. Pour un regard plus détaillé sur tous les aspects du protocole LBRY, visitez https://lbry.com/faq.

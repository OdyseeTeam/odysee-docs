# J'aime et je n'aime pas

Odysee n'a pas vraiment de préférences ni d'aversions. A la place, nous avons le feu et la boue.

De nos jours, appeler quelque chose "Feu" est un compliment. Dans de nombreux jeux/concours pour enfants des années 2000, se faire "embouer" était utilisé comme une punition.

Donc, donnez un feu à une vidéo si vous l'avez aimée. Si vous ne l'avez pas aimée, donnez-lui un slime.

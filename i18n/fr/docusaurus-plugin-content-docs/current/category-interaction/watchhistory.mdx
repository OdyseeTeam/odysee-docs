# Historique de visionnage

Après avoir regardé une vidéo, celle-ci apparaîtra dans votre historique de visionnage, qui est accessible dans la barre latérale étendue.

Cliquez sur le bouton avec trois lignes verticales dans le coin supérieur droit de l'écran pour le développer. Une fois qu'il est développé, l'historique de visionnage est le septième bouton.

Pour l'instant, l'historique de surveillance n'est pas partagé entre les appareils, ce qui signifie que les vidéos regardées sur votre ordinateur n'apparaîtront pas dans l'historique de surveillance de votre téléphone et vice versa.

Vous pouvez supprimer des vidéos spécifiques de votre historique en les survolant et en cliquant sur le "X" à droite. Vous pouvez également supprimer l'intégralité de votre historique en cliquant sur **Effacer l'historique**.

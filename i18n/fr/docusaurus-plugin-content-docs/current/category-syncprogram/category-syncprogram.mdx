# Synchronisation avec YouTube

Télécharger des centaines d'anciennes vidéos sur une nouvelle plateforme peut être une tâche ardue. Heureusement, Odysee dispose d'un [outil de synchronisation YouTube](https://odysee.com/$/youtube?reset_scroll=youtube&new_channel=true) qui permet aux créateurs de YouTube de mettre automatiquement en parallèle tous les contenus passés et futurs sur leur chaîne Odysee.

Dans cet article, nous allons passer en revue les principes de base de l'outil de synchronisation lui-même, fournir une description détaillée et répondre à quelques questions fréquemment posées.

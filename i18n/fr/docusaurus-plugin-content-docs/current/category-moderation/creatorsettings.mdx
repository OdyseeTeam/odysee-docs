## Paramètres du créateur

## Désactiver les commentaires

Le premier paramètre de la page des paramètres du créateur est une case à cocher pour autoriser les commentaires.

Si la case est cochée, les commentaires seront autorisés sur votre chaîne. Si elle n'est pas cochée, les commentaires seront désactivés sur toutes les vidéos de votre chaîne.

## Mode lent

Le deuxième paramètre de la page des paramètres du créateur est le *Mode lent*. Il définit le temps (en secondes) que les commentateurs doivent attendre entre chaque commentaire. Par exemple, si le mode lent est défini sur 10 secondes, les commentateurs devront attendre 10 secondes après avoir envoyé un commentaire pour en laisser un autre.

Cette option est utile pour les livestreams. Le mode lent permet d'éviter les spams et est souvent utilisé par les diffuseurs qui reçoivent un volume élevé de chats en livestream et ont du mal à suivre.

## Âge minimum des chaînes

Le paramètre suivant sur la page des paramètres du créateur vous permet de définir un âge minimum de la chaîne pour les commentaires. Ceci est utile pour prévenir le spam et le harcèlement.

## Pourboire minimum LBC pour les commentaires

La définition d'un pourboire minimum pour les commentaires signifie que les utilisateurs doivent laisser un pourboire d'au moins le montant indiqué pour pouvoir laisser un commentaire. Par exemple, si le montant minimum du pourboire est fixé à 10, les utilisateurs devront laisser un pourboire de 10 LBC ou plus pour pouvoir laisser un commentaire.

Cela peut être utile pour gérer les sections de commentaires. Si quelqu'un doit donner un pourboire pour pouvoir commenter, il y a de fortes chances que tous ceux qui finissent par laisser un commentaire soient des fans de votre chaîne.

## Pourboire minimum LBC pour les hyperchats

La définition d'un pourboire minimum pour les hyperchats signifie que les utilisateurs doivent laisser un pourboire d'au moins le montant indiqué pour pouvoir participer à un hyperchat. Cela n'affecte en aucun cas les discussions ou les commentaires normaux.

## Ajouter des modérateurs

L'option suivante permet d'ajouter des modérateurs. Saisissez le nom de la chaîne ou l'URL de la personne que vous souhaitez ajouter en tant que modérateur, puis sélectionnez Ajouter.

Les modérateurs délégués sont en mesure de bloquer les chaînes, de couper le son des chaînes et de supprimer les commentaires en votre nom.

## Filtrage des mots

Si vous souhaitez empêcher les utilisateurs de prononcer certains mots spécifiques dans votre section de commentaires, vous pouvez les ajouter dans la section Filtre.

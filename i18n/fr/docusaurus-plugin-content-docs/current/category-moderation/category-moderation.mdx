# Outils de modération

La plupart des plateformes médiatiques gèrent la modération de deux manières : soit la plateforme vidéo elle-même a des règles très strictes sur ce qui peut ou ne peut pas être dit dans les sections de commentaires, soit il n'y a absolument aucune règle et les commentateurs peuvent dire ce qu'ils veulent où ils veulent. 

Odysee a une approche différente. Nous croyons fermement qu'il faut permettre aux créateurs d'avoir plus de contrôle sur leurs décisions de modération dans leur espace, tant que cela reste dans le cadre des [directives de la communauté](https://help.odysee.tv/fr/communityguidelines/).

Cette section aborde les différents outils de modération dont disposent les créateurs d'Odysee pour gérer leurs propres sections de commentaires.

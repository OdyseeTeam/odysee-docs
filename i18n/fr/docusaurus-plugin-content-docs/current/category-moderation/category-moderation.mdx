# Outils de modération

La plupart des plateformes médiatiques gèrent la modération de deux manières : soit la plateforme vidéo elle-même a des règles très strictes sur ce qui peut et ne peut pas être dit dans les sections de commentaires, soit il n'y a absolument aucune règle et les commentateurs peuvent dire ce qu'ils veulent où ils veulent. 

Odysee a une approche différente. Nous croyons fermement qu'il faut laisser les créateurs décider de ce qui est ou n'est pas tolérable dans leurs propres espaces. Cela signifie qu'Odysee, en tant que plateforme, doit rester à l'écart, tout en donnant aux créateurs des outils pour traiter les mauvais acteurs comme ils l'entendent.

Cette section couvre les différents outils de modération dont disposent les créateurs d'Odysee pour gérer leurs propres sections de commentaires.

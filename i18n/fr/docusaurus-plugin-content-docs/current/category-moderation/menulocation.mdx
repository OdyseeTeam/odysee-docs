# Emplacement du menu

Le menu des paramètres du créateur est accessible en cliquant sur l'icône de l'image de profil dans le coin supérieur droit et en sélectionnant [<OdyseeIcon type="cogwheel"/> Réglages](https://odysee.com/$/settings) dans le menu déroulant.

<Image src="/img/fr/Paramètres.png" />

Sur les téléphones portables, le menu des paramètres se trouve en bas de la barre latérale étendue.

*Image*

Dans le menu des paramètres, vous ferez défiler la section **Paramètres du contenu** et sélectionnerez **Paramètres du créateur**.

*image*

# Suivre

Y a-t-il un créateur dont vous appréciez vraiment le contenu ? Vous pouvez le suivre en cliquant sur le bouton S'abonner sur sa chaîne ou sous son contenu.

Tous les créateurs que vous avez suivis apparaîtront sous *Suivis*. Cette rubrique se trouve en haut de la page d'accueil ou dans la barre latérale.

À partir de cette page, vous pouvez consulter des suggestions de nouvelles chaînes à suivre en cliquant sur le bouton **Découvrir des chaînes**.

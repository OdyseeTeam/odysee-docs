# Contenu payant

Vous avez un documentaire, un film ou un court-métrage de haute qualité que vous souhaitez proposer à la location ou à l'achat ? Vous pouvez le faire via Odysee.

Tout d'abord, vous devez vous assurer d'avoir créé un portefeuille sur Odysee. Si ce n'est pas le cas, consultez notre guide [ici](https://help.odysee.tv/fr/category-monetization/).

Lorsque vous téléchargez une vidéo via le formulaire de publication, faites défiler vers le bas jusqu'à la section **Prix**.

![Prix](https://thumbs.odycdn.com/581a9486917bc6ddc30a47f2d110b864.webp)

Vous pouvez définir le prix d'achat, le prix de location et la durée de location à l'aide du formulaire.

:::info NOTE :

L'achat ne permet pas à l'utilisateur de télécharger la vidéo. Il permet uniquement un accès illimité à la lecture de la vidéo via Odysee.

:::

Terminez le téléchargement et votre vidéo sera verrouillée derrière le prix d'achat ou de location :

![paywall](https://thumbs.odycdn.com/c69e8af67957ed1cdfd92645b100f7e5.webp)

## Acheter ou louer le contenu

Une fois que le contenu est disponible à l'achat ou à la location, il sera visible sur la page de votre chaîne et vos fans pourront l'acheter ou le louer.

![paywall](https://thumbs.odycdn.com/85ab2fd67bd79cd2190cc17e8c0ed3eb.webp)

Vous devrez confirmer l'achat du contenu via la fenêtre contextuelle du portefeuille.

![paywall](https://thumbs.odycdn.com/3340aa54ffe3b253b8295a6678ed85cb.webp)

Une fois que quelqu'un a acheté ou loué le contenu, vous recevrez une notification indiquant que votre contenu a été acheté ou loué.

![paywall](https://thumbs.odycdn.com/8b92745f809ec02fba0ef6183f344593.webp)

:::info NOTE :

N'oubliez pas que vous devez avoir un solde AR dans votre portefeuille. Si vous n'avez pas encore de solde disponible, apprenez comment acheter de l'AR sur la [page d'échange](https://help.odysee.tv/fr/category-monetization/exchanges/#purchasing-ar).

:::

# Contenu payant

Avez-vous un documentaire, un film, etc. de haute qualité que vous aimeriez louer/acheter? Vous pouvez le faire via Odysee.

Vous devez d'abord vous assurer que vous disposez d'un compte bancaire lié à Odysee. Si ce n'est pas le cas, consultez notre guide [ici](https://help.odysee.tv/category-basics/bankaccount).

Lorsque vous mettez en ligne une vidéo, faites défiler jusqu'à la section **Mots-clés**.

![](/img/en/paywall1.png)

Vous pourrez définir le prix d'achat, le prix de location et la durée de location à l'aide des mots clés.

Pour définir un prix d'achat, taper **purchase:[prix]**. Par exemple, si vous souhaitez fixer le prix d'achat à 9,99$, saisissez **purchase:[9.99]**.

Appuyez sur Entrée et vous verrez le prix sous **Mots-clés connu**.

Pour entrer le prix de location et la durée de location, vous utiliserez les mots clés encore. Taper **rental:[prix]:[temps en secondes]**. Par exemple, si vous souhaitez fixer le prix de la location à 4,99 USD et la durée de location à 24 heures, écrivez **rental:[4.99:86400]**.

Appuyez sur Entrée et vous verrez le prix et durée de location sous **Mots-clés connu**.

![](/img/en/paywall2.png)

Completez le téléchargement et votre vidéo sera disponible seulement pour le prix d'achat ou de location:

![](/img/en/paywall3.png)

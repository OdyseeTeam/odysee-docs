# Contenu payant

Vous avez un documentaire, un film, etc. de grande qualité que vous aimeriez mettre en location/achat ? Vous pouvez le faire par l'intermédiaire d'Odysee.

Vous devez d'abord vous assurer que vous disposez d'un compte bancaire lié à Odysee. Si ce n'est pas le cas, consultez notre guide [ici](https://help.odysee.tv/fr/category-basics/bankaccount).

Lorsque vous téléchargez une vidéo sur le formulaire de publication, faites défiler jusqu'à la section **Prix**.

![Price](https://thumbs.odycdn.com/581a9486917bc6ddc30a47f2d110b864.webp)

Vous pouvez définir le prix d'achat, le prix de location et la durée de location à l'aide du formulaire.

:::info NOTE:
L'achat ne permet pas à l'utilisateur de télécharger la vidéo. Il permet seulement un accès illimité à la vidéo par l'intermédiaire d'Odysee.
:::

Terminez le téléchargement, et votre vidéo sera verrouillée derrière le prix d'achat ou de location :

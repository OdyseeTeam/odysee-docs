# Suppression des téléchargements

Pour supprimer entièrement un chargement, cliquez sur l'icône de la corbeille située sous le chargement.

<Image src="/img/fr/Suppressiontéléchargements1.png" />

Vous pouvez également cliquer sur "supprimer" dans le menu à trois points sans cliquer sur le téléchargement :

<Image src="/img/fr/Suppressiontéléchargements2.png" />

:::danger
Une fois supprimé, votre téléchargement ne peut être récupéré ni par vous, ni par Odysee. Alors soyez prudent et soyez sûr de votre décision.

:::

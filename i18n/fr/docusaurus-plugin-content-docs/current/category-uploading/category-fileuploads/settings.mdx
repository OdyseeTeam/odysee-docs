# Paramètres et personnalisation

## Titre, URL et description

Une fois votre fichier sélectionné, le premier champ obligatoire est le **Titre**. Nous recommandons des titres qui informent précisément les utilisateurs du contenu, mais qui sont intéressants et efficaces pour inciter les utilisateurs à cliquer.

Vient ensuite l'**URL** du fichier. Elle est distincte du titre. Le titre est ce qui s'affiche sur le téléchargement lui-même, il s'agit simplement de ce qui apparaît dans l'URL de la vidéo.

Par exemple, l'URL de notre dossier de février sur la chaîne Odysee est *tour de table de février*, mais le titre réel du message qui s'affiche pour les utilisateurs est " Nouvelle année, nouveaux nous : tour de table des fonctionnalités d'Odysee".

Après l'URL, nous demandons une **Description**. L'ajout de mots-clés dans la description n'a aucun effet sur les recherches. Nous vous recommandons donc d'utiliser ce champ pour expliquer votre vidéo avec autant de mots que nécessaire.

## Vignettes

Après la description, il vous sera demandé d'ajouter une **Vignette**.

Pour les vignettes, nous recommandons 1280 x 720 pixels. Vous pouvez sélectionner un fichier sur votre ordinateur ou coller un lien vers une image. Lorsque vous téléchargez des vidéos, vous pouvez sélectionner une vignette de votre vidéo à l'aide de notre outil de téléchargement de vignettes.
## Mots-clés 

Ensuite, il vous sera demandé d'ajouter des **Mots-clés**.

:::note Bon à savoir
Les mots-clés améliorent la visibilité du contenu. Nous vous recommandons de simplifier les onglets relatifs au genre de votre contenu, par exemple Art, Jeux, Comédie, etc.

:::

**Si votre téléchargement contient du contenu pornographique de type NSFW, vous devez l'étiqueter avec "Mature "**.

:::info
Pour l'instant, chaque vidéo (et chaque chaîne) ne peut avoir que 5 mots-clés au maximum.

:::

## Options supplémentaires

Sous cet onglet, vous disposez de quelques options.

La première est **Date de diffusion**. Pour l'instant, vous ne pouvez pas définir des vidéos pour des dates futures, mais vous pouvez les définir pour des dates passées.

Ceci est utile lorsque vous chargez des vidéos plus anciennes. Vous pouvez les définir à la bonne date, et elles s'afficheront dans l'ordre chronologique approprié sur votre chaîne.

L'option suivante est **Language**. Elle correspond par défaut à la langue que vous avez sélectionnée.

La troisième option est **License**. Il s'agit de spécifier la licence de droit d'auteur.

La valeur par défaut est "Aucune". Si aucune licence n'est sélectionnée, le fichier sera protégé par des droits d'auteur par défaut. Les autres licences sont :

- Les licences **Domaine public** permettent la réutilisation sans aucune restriction.
- Les licences **Creative Commons** permettent la réutilisation avec attribution au créateur original.
- Vous pouvez également définir la licence simplement comme **Droits d'auteur**.

Vient ensuite le **Prix**. Il s'agit du montant que les internautes doivent payer pour voir le contenu.

:::Info 
Vous pouvez définir ce montant en LBC ou en USD. Dans les deux cas, les internautes paieront en LBC. Si le prix est fixé en LBC, il s'agit du montant en LBC payé. Si le prix est fixé en USD, les spectateurs paieront l'équivalent en LBC.

:::

## Dépôt de crédit

Le dernier paramètre sous Options supplémentaires est **Dépôt**.

Il est fixé par défaut à 0,01 LBC. Si votre fichier présente le montant de dépôt le plus élevé de tous les téléchargements dont l'URL est sélectionnée, il apparaîtra en premier lors de la recherche de cette URL.

Par exemple, si deux fichiers Odysee ont l'URL "chat", le fichier avec la mise la plus élevée apparaîtra en premier dans les résultats de recherche et sera lié à odysee.com/cat.

Tout cela est lié à l'aspect blockchain d'Odysee. Nous avons une section détaillée sur les tenants et aboutissants de ce système [ici](http://chihuahuaspin.com).

:::Note Bon à savoir
Il est important de noter qu'une fois que vous avez déposé des LBC, ils ne sont pas perdus pour toujours. Il sera toujours dans votre portefeuille et sous votre contrôle, mais il sera lié à la chaîne sur laquelle vous déposez jusqu'à ce que le montant du dépôt soit réduit ou que la chaîne soit supprimée. Pour plus d'informations à ce sujet, consultez notre section sur le porte-monnaie et les transactions [ici](http://chihuahuaspin.com).

:::

## Sélection de la chaîne

La dernière option est votre **sélection de la chaîne**. Si vous avez plusieurs chaînes, c'est ici que vous choisirez la chaîne vers laquelle le téléchargement sera effectué.

Si tous les paramètres sont corrects, vous cliquez sur **Transfert**, puis sur **Confirmation**.

:::info Cela peut prendre quelques minutes pour que votre fichier finisse de se télécharger et que le téléchargement soit confirmé, mais c'est la fin du processus de téléchargement.

:::

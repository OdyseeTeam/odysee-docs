# Conseils pour recevoir de l'argent liquide

## Connexion de votre compte bancaire

Pour recevoir un pourboire en espèces, vous devez ajouter un compte bancaire.

Pour ajouter un compte bancaire, vous devez d'abord ouvrir votre <PrimaryOdyseeButton url="https://odysee.com/$/wallet"><OdyseeIcon type="credits"/> porte-monnaie</PrimaryOdyseeButton>. Votre porte-monnaie se trouve en haut à droite de l'écran, dans la barre de menu.

<Image src="/img/fr/Ajouteruncomptebancaire01.png" />

Sur la version mobile, vous le trouverez en bas du menu latéral.

<Image src="/img/fr/Mportefeuille.png" />

De là, vous sélectionnerez **Comptes bancaires**.

<Image src="/img/fr/Ajouteruncomptebancaire02.png" />

Cela vous amènera ici, où vous sélectionnerez <PrimaryOdyseeButton>Connectez votre compte bancaire</PrimaryOdyseeButton>.

<Image src="/img/fr/Ajouteruncomptebancaire03.png" />

Cela vous permet de sortir d'Odysee et d'accéder à Stripe.

<Image src="/img/fr/Ajouteruncomptebancaire04.png" />

À partir de là, il vous suffit d'entrer les informations demandées.

## Recevoir des pourboires en espèces

Une fois que votre compte bancaire est ajouté et que vous recevez des pourboires, vous pourrez suivre le montant que vous avez reçu sur votre portefeuille.

:::info 
Tous les 2 ou 3 jours ouvrables, les pourboires envoyés sur votre porte-monnaie Odysee seront automatiquement encaissés et envoyés sur votre compte bancaire.

:::

# Aperçu

Les récompenses pour les vues sur Odysee donnent aux créateurs des gains LBC pour chaque vue vérifiée. Une vue vérifiée est une vue provenant d'un utilisateur vérifié. Cela signifie que nous avons vérifié qu'il s'agit d'une personne réelle, afin d'éviter les abus de récompenses. De même, pour prévenir les abus, seules les 7 premières vues d'un utilisateur vérifié chaque jour seront monétisées pour les créateurs.

## Combien est-ce que je gagne par vue ?

Pour vérifier vos gains par vue, accédez à la page des chaînes en cliquant sur l'icône de l'image de profil et en sélectionnant **Chaînes** dans le menu déroulant.

<Image src="/img/fr/Aperçu1.png" />

Vos gains par vue s'afficheront à droite de votre chaîne.

<Image src="/img/fr/Aperçu2.png" />

:::info **Note : 
Votre taux est susceptible de changer et est basé sur un certain nombre de variables, notamment le temps de visionnement moyen, le nombre moyen de vues, le type de contenu, l'engagement, l'emplacement du créateur et le prix du LBC.**

:::

## Suivi des gains

Cliquez sur l'icône de l'image de profil dans le coin supérieur droit et sélectionnez **Récompenses**.

Sur cette page, les récompenses de visionnage gagnées apparaîtront sous **Récompenses demandées** en tant que *Bonus de visionnage de contenu*.

# Envoi de pourboires LBC

Pour envoyer un pourboire, vous devez cliquer sur le bouton **soutien** sous un téléchargement.

<Image src="/img/fr/Soutien1.png" />

Cette option est également disponible sur les chaînes.

<Image src="/img/fr/soutien2.png" />

Dans ce menu, vous sélectionnerez le bouton **Pourboire** avec l'icône LBC, choisirez un montant et cliquerez sur **Envoyer le pourboire**.

Ce pourboire LBC renforcera le contenu/la chaîne pour laquelle il a été envoyé, jusqu'à ce que le créateur choisisse de le débloquer pour l'utiliser ailleurs sur Odysee.

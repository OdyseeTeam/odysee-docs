# Types de récompenses

Pour afficher la page des récompenses, cliquez sur l'icône avatar dans le coin supérieur droit de l'écran et sélectionnez **Récompenses** dans le menu déroulant.

La partie supérieure de l'écran affichera la liste des récompenses proposées. Celles-ci sont :

- **Compétition de vues** : Cette récompense est attribuée en fonction du nombre total de vidéos regardées depuis que vous avez rejoint Odysee. Pour éviter les abus, seules 10 vidéos par jour seront comptabilisées.
- **Créateur d'images** : Cette récompense est attribuée en fonction du nombre total de followers vérifiés (followers avec des comptes vérifiés) de votre chaîne.
- **Le voyage** : Cette récompense est donnée pour les créateurs qui suivent.
- Récompense pour le visionnage hebdomadaire** : Cette récompense est attribuée pour avoir regardé des vidéos pendant au moins 3 jours dans une semaine donnée.
- **Les cinq favoris** : Cette récompense est attribuée pour avoir reposté le contenu de 5 créateurs différents.
- **Code personnalisé** : Si vous avez reçu un code de récompense personnalisé d'un créateur, c'est ici que vous pouvez l'entrer pour LBC.
- Invitations** : Vous pouvez gagner des LBC simplement en invitant d'autres personnes sur Odysee. Pour plus d'informations, cliquez sur la section suivante.

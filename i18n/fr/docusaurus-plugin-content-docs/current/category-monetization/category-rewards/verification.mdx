# Vérification

La vérification vous permet de gagner des crédits LBRY (LBC en abrégé) simplement en utilisant Odysee. Il s'agit de notre programme de récompenses. Nous exigeons une vérification pour gagner des récompenses afin d'éviter les abus.

La plupart des comptes sont vérifiés automatiquement. En fonction de facteurs tels que l'emplacement, l'utilisation du VPN, etc., il peut vous être demandé des informations supplémentaires pour la vérification. Pour ce faire, vous avez plusieurs options :

:::info **Par le biais du numéro de téléphone**

:::

Votre première option est de vérifier par le biais de votre téléphone. Nous vous enverrons un message texte avec un code.

:::info **Par carte**

:::

Vous pouvez également vérifier en utilisant une carte de crédit ou de débit. Nous n'enregistrerons pas les informations de votre carte et ne vous facturerons pas de frais. Il se peut que vous voyiez une autorisation de 1 $, mais celle-ci disparaîtra.

:::info **Par chat**

:::

Enfin, vous pouvez vérifier par le biais de notre serveur Discord. Ici, un modérateur examinera votre compte et, si vous êtes admissible, vous approuvera pour les récompenses. Notre Discord est ici.

Pour l'instant, les demandes de vérification sont examinées manuellement, ce qui signifie que leur approbation peut prendre de quelques heures à plusieurs jours.

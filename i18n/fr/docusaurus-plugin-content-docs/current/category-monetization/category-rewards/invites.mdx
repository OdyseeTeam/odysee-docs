# Invitations

Vous pouvez gagner des LBC simplement en invitant d'autres personnes à Odysee. Sur <a href="https://odysee.com/$/invite"><OdyseeIcon type="Users"/> Invitations</a>  vous recevrez un lien d'invitation.

Vous pouvez également accéder à cette page en cliquant sur l'icône de l'avatar dans le coin supérieur droit de l'écran et en sélectionnant **Invitations** dans le menu déroulant.

![](/img/fr/Invitation.png)

Partagez ce lien, et pour chaque utilisateur unique qui clique dessus, vous recevrez un peu de LBC.

:::info

C'est particulièrement utile pour les créateurs qui partagent leur chaîne Odysee avec leurs téléspectateurs. Au lieu de donner le lien de base de votre chaîne, donnez le lien d'invitation.

:::

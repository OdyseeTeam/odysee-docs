# Visualisation des listes de lecture

## Vos listes de lecture

Vous pouvez afficher vos propres listes de lecture en cliquant sur le bouton **Listes de lecture** dans la barre latérale.

<Image src="/img/fr/Listedelecture.png" />

Toutes vos listes de lecture, publiées et non publiées, apparaîtront dans ce menu.

## Listes de lecture d'autres chaînes

Pour afficher les listes de lecture d'une autre chaîne, vous devez d'abord vous rendre sur la page de cette chaîne.

À partir de là, sélectionnez l'onglet **Listes de lecture**. 

<Image src="/img/fr/Listedelecture2.png" />

Toutes les listes de lecture publiques créées par cette chaîne apparaissent ici. 

:::info 

Les listes de lecture qui n'ont pas été rendues publiques n'apparaîtront pas ici et ne pourront être consultées que par la chaîne qui les a créées.

:::

# Création de listes de lecture

Pour créer une nouvelle liste de lecture, cliquez d'abord sur le bouton **Listes de lecture** dans la barre latérale.

Dans ce menu, vous cliquerez sur **Créer une liste de lecture**. Il suffit de lui donner un nom et de cliquer sur la coche pour qu'elle soit créée.

Tant qu'elles ne sont pas publiées, les listes de lecture créées ne sont visibles que par vous.

## Ajout de contenu

Pour ajouter une vidéo à une liste de lecture, vous avez plusieurs possibilités.

Lorsque vous ne visualisez pas une vidéo, cliquez sur le menu à trois points, sélectionnez **Ajouter à la sélection**, puis sélectionnez la sélection à laquelle vous souhaitez ajouter la vidéo.

Lorsque vous visualisez une vidéo, cliquez sur le bouton **Enregistrer** sous la vidéo et sélectionnez la liste de lecture à laquelle vous souhaitez ajouter la vidéo.

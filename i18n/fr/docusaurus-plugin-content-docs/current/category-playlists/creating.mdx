# Création de listes de lecture

Pour créer une nouvelle liste de lecture, cliquez d'abord sur le bouton <a href="https://odysee.com/$/playlists"><OdyseeIcon type='Playlist'/> Liste de lecture</a> dans la barre latérale.

<Image src="/img/fr/Listedelecture.png" />

Dans ce menu, vous cliquerez sur <PrimaryOdyseeButton>New Playlist</PrimaryOdyseeButton>. 

<Image src="/img/fr/Créerdeslistesdelecture1.png" />

Il suffit de lui donner un nom et de cliquer sur la coche pour qu'elle soit créée.

<Image src="/img/fr/Créerdeslistesdelecture2.png" />

Tant qu'elles ne sont pas publiées, les listes de lecture créées ne sont visibles que par vous.

## Ajout de contenu

Pour ajouter une vidéo à une liste de lecture, vous avez plusieurs possibilités.

Lorsque vous ne visualisez pas une vidéo, cliquez sur le menu à trois points, sélectionnez **Add to playlist**, puis sélectionnez la sélection à laquelle vous souhaitez ajouter la vidéo.

<Image src="/img/fr/Créerdeslistesdelecture3.png" />

Lorsque vous visualisez une vidéo, cliquez sur le bouton **Sauver** sous la vidéo et sélectionnez la liste de lecture à laquelle vous souhaitez ajouter la vidéo.

<Image src="/img/fr/Créerdeslistesdelecture4.png" />

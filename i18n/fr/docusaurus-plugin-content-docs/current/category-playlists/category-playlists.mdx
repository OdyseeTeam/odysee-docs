# Listes de lecture

Il y a beaucoup de vidéos sur Odysée, et un excellent moyen de les organiser consiste à utiliser des listes de lecture. Qu'il s'agisse de créateurs créant des listes de lecture pour leur propre contenu ou des listes de lecture pour le contenu d'autres personnes, les listes de lecture ont de nombreuses utilisations.

Cette section couvrira tout ce qui concerne les listes de lecture sur Odysee.

# Publication des listes de lecture

Pour que votre liste de lecture puisse être consultée par d'autres personnes, vous devez la publier.

Pour publier une liste de lecture, accédez d'abord au menu des listes de lecture en cliquant sur le bouton **Listes de lecture** dans la barre latérale.

Passez la souris sur la liste de lecture que vous souhaitez publier, cliquez sur le menu à trois points sur la droite, puis sélectionnez **Publier**.

Dans ce menu, vous pouvez ajouter des éléments tels que des vignettes, des balises et une description, mais tout ce qui est nécessaire est un nom et une chaîne sélectionnée.

Si vous avez sélectionné la chaîne sur laquelle vous souhaitez publier et un nom qui vous convient, cliquez sur **Publier** en bas de l'écran.

Après la confirmation de la publication, votre liste de lecture apparaîtra sur votre chaîne sous l'onglet Listes de lecture, où tout le monde peut la consulter.

**Remarque : la publication porte sur la version actuelle de votre liste de lecture. Si vous y apportez des modifications à l'avenir, par exemple en ajoutant ou en supprimant des vidéos, vous devrez publier à nouveau pour que ces changements soient rendus publics**.

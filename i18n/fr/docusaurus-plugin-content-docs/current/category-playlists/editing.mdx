# Modification des listes de lecture

Pour modifier une liste de lecture, accédez d'abord au menu des listes de lecture en cliquant sur le bouton <a href="https://odysee.com/$/playlists"><OdyseeIcon type="Playlist"/> Liste de lecture</a> dans la barre latérale.

![](/img/fr/Listedelecture.png)

Passez la souris sur la liste de lecture que vous souhaitez publier, cliquez sur le menu à trois points sur la droite, puis sélectionnez **Editer**.

À partir de là, vous pouvez modifier le titre, la vignette et la description sous **Général**.

Sous **Éléments**, vous pouvez modifier les vidéos de la sélection.

- Supprimez des vidéos en cliquant sur l'icône de la corbeille.
- Déplacez une vidéo vers le haut ou le bas en cliquant sur les flèches situées directement à côté de l'icône de la corbeille.
- Envoyez les vidéos en haut ou en bas en cliquant sur les flèches avec des lignes à côté d'elles.
- Le bouton situé tout à gauche vous permet de faire glisser les vidéos dans la liste de lecture. Cliquez et maintenez le bouton enfoncé, puis déplacez la vidéo où vous le souhaitez.

Après avoir effectué les modifications, cliquez sur le bouton <PrimaryOdyseeButton>Soumettre</PrimaryOdyseeButton> en bas de l'onglet Général.

:::info Remarque 
Vous devez publier votre liste de lecture pour que ces modifications soient rendues publiques.

:::

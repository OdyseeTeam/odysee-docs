# Hiperchats y propinas para comentarios

## Hiperchats

Los hiperchats son chats destacados durante las transmisiones en vivo. Se envían adjuntando una propina.

Después de escribir tu mensaje, puedes enviar un hiperchat seleccionando el ícono del dólar debajo del cuadro de chat.

Aquí, selecciona una cantidad y pulsa **Enviar**.

Tu hiperchat se resaltará y mostrará la cantidad de la propina.

![hyperchats](https://thumbs.odycdn.com/5cb27970680ea458b6704ce98729c20f.webp)

## Propinas para comentarios

Las propinas para comentario se envían adjuntando una propina a un comentario.

Después de escribir tu comentario, puedes enviar la propina seleccionando el icono del dólar debajo del cuadro de comentarios.

Aquí, selecciona una cantidad y pulsa **Enviar**.

Tu comentario mostrará la cantidad de la propina.

# Cancelar una Membresía

Para cancelar una membresía, navegue hasta el canal al que se unió y seleccione la pestaña de membresía.

![](https://i.imgur.com/vjnKkRD.png)

También puede navegar a esta pantalla a través del portal de donantes.

Simplemente haga clic en el ícono de ajustes y seleccione Cancelar membresía, y su membresía será cancelada.

![](https://i.imgur.com/0pZGd6T.png)

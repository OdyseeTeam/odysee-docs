## ¿Por qué usar el Odysee Sync Tool (OST)?

El **Odysee Sync Tool (OST)** está diseñado para creadores que necesitan más flexibilidad. Es la solución ideal cuando la sincronización automática falla o no cumple con los requisitos para la sincronización automática.

Con [**Odysee Sync Tool (OST)**](https://sync.odysee.tv), puedes ejecutarlo en tu PC de escritorio (Windows/Mac/Linux) para automatizar el proceso por ti. Es muy fácil de usar, solo un par de clics, y tu PC descargará el contenido de YouTube y lo subirá a Odysee.

## ¿Por qué es importante?

- **Migración sin bloqueos:** Permite transferir contenido de tus canales que pueden estar inactivos o que simplemente no cumplen con los requisitos para la sincronización automática.
- **Evitar límites:** Permite sincronizar contenido que no se encuentra dentro de los límites predeterminados, como videos excepcionalmente largos.
- **Archivo completo:** Asegura que puedas llevar todo tu catálogo de videos a Odysee. Usa esta herramienta para tener control total sobre tu proceso de migración.

### Por favor descarga aquí:

**Windows:** https://sync.odysee.tv/public/OdyseeSync.exe

**Mac/Linux:** https://sync.odysee.tv

**Nota:** Recomendamos configurar la opción de **Cookies** con **Firefox** si los videos fallan, las razones típicas son tener un alto volumen de videos o contenido con restricción de edad.

:::tip Importante

- **Cookies del navegador:** Para canales con muchos videos, es posible que necesites usar la opción "Use browser cookies" (clic derecho > preferences desde la barra de tareas).
  - Hemos comprobado que **Firefox** funciona mejor en Windows.
  - Te recomendamos descargar Firefox, iniciar sesión en YouTube desde ahí y luego activar esta opción.
  - Esto también funciona bien para videos con restricción de edad.
- **Reintentos:** Para volver a intentar cualquier fallo, por ahora tendrás que cerrar y reiniciar la aplicación.
- **Nuevos usuarios:** Odysee Sync Tool (OST) creará un canal nuevo para ti.
- **Canales ya sincronizados:** Si has subido contenido manualmente mientras la sincronización estaba desactivada, el contenido **no** se deduplicará.

:::

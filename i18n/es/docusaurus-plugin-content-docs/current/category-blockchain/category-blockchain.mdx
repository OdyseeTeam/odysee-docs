# La blockchain de Odysee

Uno de los principales diferenciadores de Odysee de otras plataformas de medios es que Odysee se basa en una cadena de bloques (blockchain) llamada LBRY.

En esta sección, exploraremos qué significa exactamente eso y cómo afecta a Odysee. Es importante tener en cuenta que nadie tiene que saber nada sobre cadenas de bloques (blockchain) para usar y disfrutar de Odysee. Es similar a conducir un automóvil, puedes conducir uno sin entender cómo funciona el motor.

Esta sección se centrará principalmente en cómo Odysee interactúa con el protocolo LBRY. Para obtener una visión más detallada de cada aspecto del protocolo LBRY, visite https://lbry.com/faq.

# Verificación

No se requiere verificación para usar Odysee, pero es muy recomendable. La verificación le permite ganar Créditos LBRY (LBC para abreviar) simplemente usando Odysee. Requerimos verificación para recibir LBC, y de esta manera evitar abusos.

La mayoría de las cuentas se verifican automáticamente. Dependiendo de factores como la ubicación, el uso de VPN, etc., es posible que se le solicite información adicional para la verificación. Para ello, tienes algunas opciones:

**A través del número de teléfono**

Su primera opción es verificar a través de su número de teléfono. Te enviaremos un mensaje de texto con un código. No todas las regiones son aptas para la verificación por número de teléfono.

**A través de tarjeta**

También puede verificar usando una tarjeta de crédito o débito conectada a una cuenta bancaria en dólares. Odysee no guarda la información de su tarjeta ni le cobraremos ninguna tarifa. Es posible que vea una autorización de pago de $1 en su cuenta bancaria, pero esto regresará a su cuenta.

**A través de chat**

Finalmente, puede verificar a través del servidor de Discord. Aquí, un moderador revisará su cuenta y, si es elegible, aprobará sus recompensas. Ingresa al Discord de Odysee [aquí](https://discord.com/invite/sWedwsNWjX).

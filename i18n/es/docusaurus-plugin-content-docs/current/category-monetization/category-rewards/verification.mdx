# Verificación

La verificación le permite ganar Créditos LBRY (LBC para abreviar) simplemente usando Odysee. Este es nuestro programa de recompensas. Requerimos verificación para ganar recompensas para evitar abusos.

La mayoría de las cuentas se verifican automáticamente. Dependiendo de factores como la ubicación, el uso de VPN, etc., es posible que se le solicite información adicional para la verificación. Para ello, tienes algunas opciones:

**A través del número de teléfono**

Su primera opción es verificar a través de su teléfono. Te enviaremos un mensaje de texto con un código.

**A través de tarjeta**

También puede verificar usando una tarjeta de crédito o débito. No guardaremos la información de su tarjeta ni le cobraremos ninguna tarifa. Es posible que vea una autorización de $1, pero regresará a su tarjeta.

**A través de chat**

Finalmente, puede verificar a través de nuestro servidor Discord. Aquí, un moderador revisará su cuenta y, si es elegible, aprobará sus recompensas. Nuestro Discord está aquí.

En este momento, las solicitudes de verificación se revisan manualmente, lo que significa que pueden tardar desde unas pocas horas hasta varios días en aprobarse.

# Verificación

No se requiere verificación para usar Odysee, pero es muy recomendable. La verificación le permite recibir algunos Créditos simplemente usando Odysee. Requerimos la verificación para que pueda recibir algunos Créditos, y así para evitar abusos.

La mayoría de las cuentas se verifican automáticamente. Dependiendo de factores como la ubicación, el uso de VPN, etc., es posible que se le solicite información adicional para la verificación. Para hacer esto, tienes algunas opciones:

![odysee](https://thumbs.odycdn.com/2e872f027ae93478191fea0c104b7bbc.webp)

## Verificar mediante tarjeta de crédito

También puede verificar utilizando una tarjeta de crédito o débito conectada a una cuenta bancaria en dólares. Odysee no almacena la información de su tarjeta ni le cobraremos ninguna tarifa. Es posible que vea una autorización de pago de $1 en su cuenta bancaria, pero se devolverá a su cuenta.

## Verificar por correo electrónico

Puede solicitar la verificación de su cuenta en Odysee enviando un correo electrónico a hello@odysee.com.

:::info Nota

En este momento, las solicitudes de verificación se revisan manualmente, lo que significa que pueden tardar desde unas pocas horas hasta varios días en aprobarse.

:::

# Envío de propinas LBC

Para enviar una propina, haga clic en el botón **Apoyar** debajo del contenido.

![](/img/es/image_42.png)

Esta opción también está disponible en los canales.

![](/img/es/image_43.png)

Desde este menú, seleccionará el botón **Propina** con el ícono LBC, seleccionará una cantidad y hará clic en **Enviar propina**.

Esta propina de LBC impulsará el contenido/canal al cual está apoyando, hasta que el creador decida desbloquearlo para usarlo en otro lugar de Odysee.
